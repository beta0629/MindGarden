# MindGarden 예시 메뉴 구조

**작성일**: 2025-11-20  
**버전**: 1.0.0  
**상태**: 완료

---

## 📋 개요

MindGarden은 CONSULTATION 업종의 실제 구현 예시입니다. 이 문서는 MindGarden의 메뉴 구조를 상세히 정리한 것입니다.

**⚠️ 참고**: MindGarden의 메뉴 구조는 CONSULTATION 업종의 예시일 뿐이며, 다른 상담소는 다른 메뉴 구조를 가질 수 있습니다.

---

## 👤 역할별 메뉴 구조

### 1. 내담자 (CLIENT) 메뉴

```
대시보드
  └─ 홈

상담 예약
  ├─ 상담사 선택
  ├─ 일정 선택
  └─ 예약 확인

상담 내역
  ├─ 예정된 상담
  ├─ 완료된 상담
  └─ 취소된 상담

상담 메시지
  └─ 메시지 목록

결제 내역
  ├─ 결제 목록
  └─ 결제 상세

프로필
  ├─ 내 정보
  ├─ 비밀번호 변경
  └─ 알림 설정
```

**메뉴 접근 권한:**
- COMMON_MENU: ✅
- CLIENT_MENU: ✅

---

### 2. 상담사 (CONSULTANT) 메뉴

```
대시보드
  └─ 홈

일정 관리
  ├─ 캘린더 보기
  ├─ 일정 추가
  └─ 일정 수정

할당된 내담자
  ├─ 내담자 목록
  └─ 내담자 상세

상담 기록
  ├─ 상담 기록 작성
  └─ 상담 기록 조회

상담 통계
  └─ 자신의 상담 통계

프로필
  ├─ 내 정보
  └─ 비밀번호 변경
```

**메뉴 접근 권한:**
- COMMON_MENU: ✅
- CONSULTANT_MENU: ✅

---

### 3. 관리자 (ADMIN) 메뉴

```
대시보드
  └─ 홈

사용자 관리
  ├─ 상담사 관리
  │   ├─ 상담사 목록
  │   ├─ 상담사 추가
  │   └─ 상담사 상세
  ├─ 내담자 관리
  │   ├─ 내담자 목록
  │   ├─ 내담자 추가
  │   └─ 내담자 상세
  └─ 사용자 관리
      └─ 전체 사용자

일정 관리
  ├─ 전체 일정 조회
  └─ 일정 관리

통계
  ├─ 전체 통계
  ├─ 상담 통계
  └─ 리포트

결제 관리
  ├─ 결제 내역
  └─ 정산

매핑 관리
  ├─ 상담사-내담자 매핑
  └─ 회기 관리

설정
  ├─ 공통코드 관리
  │   ├─ CoreSolution 공통 코드 (읽기 전용)
  │   └─ MindGarden 공통 코드 (읽기/쓰기)
  └─ 시스템 설정
```

**공통 코드 관리 메뉴 상세:**
```
공통코드 관리
├─ 코드 그룹 선택
│   ├─ CoreSolution 공통 코드
│   │   ├─ 코드 목록 (읽기 전용)
│   │   └─ 코드 상세 조회
│   └─ MindGarden 공통 코드
│       ├─ 코드 목록
│       ├─ 코드 추가
│       ├─ 코드 수정
│       └─ 코드 삭제
└─ 코드 타입 필터
    ├─ CORE (CoreSolution 코드)
    └─ TENANT (입점사 코드)
```

**메뉴 접근 권한:**
- COMMON_MENU: ✅
- ADMIN_MENU: ✅

**공통 코드 권한:**
- CoreSolution 공통 코드: 읽기 전용 (HQ Admin만 수정 가능)
- MindGarden 공통 코드: 읽기/쓰기 (Tenant Admin 수정 가능)

---

### 4. 지점 수퍼 관리자 (BRANCH_SUPER_ADMIN) 메뉴

```
관리자 메뉴의 모든 항목
  +
ERP 관리
  ├─ ERP 대시보드
  ├─ 구매 요청
  ├─ 관리자 승인
  ├─ 아이템 관리
  ├─ 예산 관리
  └─ 주문 관리
  +
ERD 메뉴
  └─ ERD 조회
```

**메뉴 접근 권한:**
- COMMON_MENU: ✅
- ADMIN_MENU: ✅
- BRANCH_SUPER_ADMIN_MENU: ✅ (ERP 메뉴 포함)

---

### 5. 본사 관리자 (HQ_ADMIN) 메뉴

```
대시보드
  └─ 홈

테넌트 관리
  ├─ 테넌트 목록
  └─ 테넌트 상세

통계
  └─ 전체 테넌트 통계

설정
  └─ 시스템 설정
```

**메뉴 접근 권한:**
- COMMON_MENU: ✅
- HQ_ADMIN_MENU: ✅

---

### 6. 본사 총관리자 (HQ_MASTER) 메뉴

```
모든 메뉴 접근 가능
  ├─ COMMON_MENU
  ├─ ADMIN_MENU
  ├─ HQ_ADMIN_MENU
  ├─ BRANCH_SUPER_ADMIN_MENU (ERP 메뉴 포함)
  ├─ CONSULTANT_MENU
  └─ CLIENT_MENU
```

**ERP 관리 메뉴:**
```
ERP 관리
  ├─ ERP 대시보드
  ├─ 구매 요청
  ├─ 관리자 승인
  ├─ 수퍼 관리자 승인
  ├─ 아이템 관리
  ├─ 예산 관리
  └─ 주문 관리
```

**메뉴 접근 권한:**
- 모든 메뉴 그룹: ✅

## 🔄 메뉴 계층 구조

### 메인 메뉴 → 서브 메뉴

```
사용자 관리 (메인)
  ├─ 상담사 관리 (서브)
  │   ├─ 상담사 목록 (페이지)
  │   ├─ 상담사 추가 (페이지)
  │   └─ 상담사 상세 (페이지)
  ├─ 내담자 관리 (서브)
  │   ├─ 내담자 목록 (페이지)
  │   ├─ 내담자 추가 (페이지)
  │   └─ 내담자 상세 (페이지)
  └─ 사용자 관리 (서브)
      └─ 전체 사용자 (페이지)
```

## ⚠️ 중요 사항

1. **MindGarden은 예시일 뿐**
   - MindGarden의 메뉴 구조는 CONSULTATION 업종의 예시
   - 다른 상담소는 다른 메뉴 구조를 가질 수 있음

2. **동적 메뉴 시스템**
   - 메뉴는 CommonCode 기반으로 동적으로 관리
   - 테넌트 관리자가 메뉴 커스터마이징 가능

3. **역할별 다른 메뉴**
   - 각 역할은 자신의 권한에 맞는 메뉴만 표시
   - 메뉴 접근 권한은 데이터베이스에서 관리
