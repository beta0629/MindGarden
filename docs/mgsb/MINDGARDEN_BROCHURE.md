# MindGarden 통합 상담·비즈니스 플랫폼 브로셔

> 상담 중심 비즈니스 운영의 모든 순간을 하나의 플랫폼으로 연결합니다.

## 1. 플랫폼 개요

- **대상**: 상담센터, 학원, 프랜차이즈, 소규모 서비스 사업자를 위한 종합 관리 솔루션
- **핵심 가치**: 회원·상담·예약·결제·정산·알림을 한 화면에서 관리하는 All-in-One 플랫폼
- **운영 기반**: 멀티테넌시 구조와 ERP 수준의 정산 시스템을 갖춘 확장형 아키텍처
- **비전**: “상담 기반 비즈니스의 디지털 전환을 한 번에”

## 2. 주요 특장점

### 2.1 통합 상담·예약 관리

- 상담 예약 → 상담 기록 → 수강/서비스 등록까지 끊김 없는 흐름 제공
- 웹/모바일 채널에서 예약 접수 후 관리자가 실시간 승인·조정 가능
- 방문 상담, 온라인 상담, 대면/비대면 수업까지 유연하게 지원
- 상담 이력, 담당자 배정, 일정 충돌 알림 기능으로 운영 안정성 향상

### 2.2 실시간 정산·ERP 연동

- PL/SQL 기반 배치로 매출·수수료·정산금 자동 계산, 오류 자동 감지
- 관리자 포털에서 정산 리포트(PDF/CSV) 다운로드, ERP·회계 시스템 연동
- 테넌트별 수익/비용 분석, 요일/시간대별 매출 리포트 제공
- 부분 환불, 재처리, 이중 결제 검증 기능으로 금융 리스크 최소화

### 2.3 강력한 멀티테넌시 & 권한 제어

- 지점/테넌트별 데이터 완전 분리, 테넌트 커스텀 브랜딩 지원
- HQ(본사), 지점 관리자, 스태프, 상담사, 고객 역할별 세밀한 접근 제어
- HQ 대시보드에서 전국 지점 KPI 모니터링, 정책 공지 일괄 발송
- 감사 로그, 세션 추적, 역할 기반 메뉴 노출로 보안과 편의 동시 확보

### 2.4 통합 알림 & 메시징 시스템

- 상담 확정, 결제, 공지, 이벤트 등 전체 라이프사이클 자동 알림 발송 (SMS/이메일/웹)
- NotificationContext로 공개 페이지 접근 시 불필요한 API 호출 차단, 401 오류 예방
- 읽음/읽지 않은 상태, 이벤트별 템플릿, 발송 로그 관리와 재발송 자동화
- 향후 카카오 알림톡, 푸시 메시지 등 확장 플러그인의 추가가 용이한 구조

### 2.5 소비자 & 관리자 포털 제공

- **소비자 포털**: 학원·상담 서비스 탐색, 예약, 결제, 영수증, 일정 확인, 쿠폰/프로모션 수신
- **관리자 포털**: 대시보드, 예약/수강 관리, CRM, 알림, 통계, 정산 리포트, 출결 관리
- **모바일 최적화**: 상담사는 태블릿으로 출결/상담 메모, 소비자는 모바일로 결제/영수증 확인
- **브랜딩 지원**: 테넌트별 로고/색상/도메인 커스터마이징으로 브랜드 일관성 유지

### 2.6 MindGarden Design System v2.0

- 반응형 UI와 접근성 표준(WCAG 2.1 AA) 준수, 다크모드까지 고려한 토큰 설계
- Storybook 기반 컴포넌트 라이브러리, CSS 변수(`mg-`) 체계로 재사용성과 유지보수성 확보
- 업종별 UI 키트를 모듈화하여 빠른 커스터마이징 지원 (학원, 요식업, 피트니스 등)
- 시각적 회귀 테스트(Percy/Chromatic)로 디자인 품질 보장

### 2.6 모바일 앱(Android / iOS) 지원

- 기존 웹 포털 기능을 그대로 모바일 앱으로 확장 가능 (React Native 기반 하이브리드 전략 검토 완료)
- 상담사/지점 스태프는 태블릿·스마트폰에서 출결, 상담 기록, 알림, 일정 관리를 수행
- 소비자는 모바일 앱으로 상담 예약, 결제, 영수증 확인, 쿠폰 수신, 푸시 알림 확인
- 오프라인 매장(학원/센터)에서도 Wi-Fi/셀룰러 네트워크로 실시간 동기화
- 향후 앱스토어/플레이스토어 공식 배포 계획 및 모바일 전용 UI 최적화 로드맵 수립

### 2.7 데이터 보안 & 컴플라이언스

- 개인정보 암호화, 세션/권한 체계 표준화, OWASP 기반 보안 검사 수행
- 국내 개인정보보호법, 전자금융거래법 준수를 위한 로그 추적 및 데이터 보존 정책
- API 접근 제어, 레이트 리밋, SSL/TLS 암호화, DB 접근 권한 관리 강화
- 정기 보안 점검 및 침해사고 대응 프로세스를 체계적으로 운영

## 4. 확장성과 통합 전략

### 4.1 유연한 업종 확장

- 학원/교육, 요식업, 피트니스, 미용/케어, 프랜차이즈 본사 등 다양한 업종으로 확장 가능
- 업종별 플러그인 구조로 고객 요구에 따라 기능을 선택적으로 활성화
- 각 산업군의 핵심 지표와 업무 프로세스를 표준화한 Blueprint 제공

### 4.2 외부 연동 & 파트너십

- 결제 게이트웨이(PG), ERP, 회계, 세무, CRM, 메신저 등 서드파티 시스템과 API 연동
- 향후 배달 플랫폼, 채팅 상담, LMS, HR 시스템과의 통합도 용이
- REST/GraphQL API, Webhook, 메시지 큐(Kafka) 등 다양한 통신 방식 지원

### 4.3 DevOps & 모니터링

- GitHub Actions 기반 CI/CD → QA → 운영 배포 자동화
- APM, 로그, 슬랙·이메일 알림으로 24/7 운영 모니터링
- 멀티 리전/멀티 존 배포 전략과 백업/복구 시나리오 마련

## 5. QA & 성능 보증

- Playwright/Cypress 기반 E2E 테스트, Percy 시각적 회귀, Axe 접근성 자동화
- JMeter·Gatling으로 API SLA, 정산 배치(10만 건) 부하 테스트
- PL/SQL 정산 테스트 시나리오 및 재처리 검증 프로세스 확보
- QA 체크리스트, 릴리즈 게이트, UAT 프로토콜 정립으로 품질 보장

## 6. 고객 지원 & 운영 서비스

- **온보딩 지원**: 테넌트 등록, 데이터 이관, 교육 프로그램, 운영 가이드 제공
- **고객 지원 채널**: 헬프데스크, 챗봇, 긴급 대응 핫라인, SLA 기반 지원
- **지속 업데이트**: 월간 릴리즈 노트, 기능 개선 로드맵, 사용자 커뮤니티 운영
- **컨설팅 서비스**: 업무 프로세스 진단, 커스터마이징, 데이터 분석 컨설팅

## 7. 도입 기대 효과

- 상담/교육 비즈니스 운영 효율 극대화, 수기 업무 60% 이상 절감
- 빠른 상담 응답, 간편 결제, 투명한 정산으로 고객 만족도 향상
- HQ와 지점 간 실시간 데이터 공유로 조직 내 의사결정 속도 향상
- 멀티 업종 SaaS로 확장 가능한 코어 플랫폼 확보, 신규 수익 모델 창출

## 8. 차별화 포인트 요약

- 상담 특화 ERP 경험을 기반으로 한 정산/알림/예약 통합 플랫폼
- 멀티테넌시, 보안, QA 체계를 고려한 엔터프라이즈급 설계
- 업종 확장성과 커스터마이징이 가능하며 빠르게 론칭 가능한 플랫폼
- 디자인 시스템, DevOps, QA, PL/SQL 배치 등 플랫폼 운영에 필요한 모든 요소 준비
