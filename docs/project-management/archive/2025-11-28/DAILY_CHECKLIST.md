- [ ] 서버 실행 오류 → 포트 충돌 확인, 프로세스 종료 후 재시작

### **코드 구현 문제**
- [ ] 타입스크립트 오류 → 타입 정의 확인, any 타입 임시 사용
- [ ] React 훅 오류 → 의존성 배열 확인, 무한 루프 방지
- [ ] API 연동 오류 → CORS 설정, 엔드포인트 URL 확인

### **성능 문제**
- [ ] 메모리 누수 → 이벤트 리스너 정리, useEffect cleanup
- [ ] 렌더링 지연 → React.memo 사용, 불필요한 리렌더링 방지
- [ ] 네트워크 지연 → 캐싱 전략, 요청 최적화
