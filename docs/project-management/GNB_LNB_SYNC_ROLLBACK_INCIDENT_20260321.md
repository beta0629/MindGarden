## 1. 요약

- **증상**: 관리자에서 `/admin/dashboard` 등 일부는 동작하나, **LNB로 진입하는 다른 관리 화면**에서 **하얀 화면** 또는 **`Minified React error #130`**(객체를 JSX 자식으로 렌더) 류 오류가 재현됨.
- **조치**: GNB·LNB **경로/상태 정합**을 목표로 한 변경이 원인 후보로 올라와, 해당 범위를 **커밋 단위로 Revert**하여 **싱크 적용 이전 상태**로 되돌림.
- **결과**: 롤백 배포 후 **화면 정상 표시** 확인(현장 피드백).

## 4. 롤백 시 유의사항

- **`b434862ae` Revert** 과정에서 `gnbQuickActions.js` 등은 **충돌 해소를 위해 `79a8a3640` 시점 내용**을 기준으로 맞춤.
- **`docs/project-management/GNB_LNB_MENU_SYNCHRONIZATION_DIRECTIVE.md`** 는 `b434862ae`에서 추가된 문서로, Revert로 **저장소에서 제거**됨. 필요 시 별도 문서로 원칙만 재작성 가능.
- 이후 **`cdadf5a73` 등 후속 커밋**에서 조정했던 상담사 퀵액션(navigate vs modal) 등은 **싱크 이전 `gnbQuickActions` 구조**에 맞춰져 있으므로, 향후 “복구” 시 **작은 PR**로만 재적용하는 것이 안전하다.
