## AI 이미지 생성 도구별 사용법

### 1. Midjourney
```
/imagine prompt: [프롬프트] --ar 21:4 --v 6 --style raw
```
- `--ar 21:4`: 배너 비율 (가로로 긴 형태)
- 또는 `--ar 16:3` (더 넓은 배너)

### 2. DALL-E 3 (ChatGPT)
- 프롬프트 입력
- 설정: 1792x1024 또는 커스텀 비율
- "배너 이미지, 가로형" 추가

### 3. Stable Diffusion
- 프롬프트 입력
- Negative prompt: "dark, scary, cluttered, messy, text, logo, watermark, people"
- 설정: 1920x400, steps: 30-50
- Aspect ratio: 21:4 또는 16:3

### 4. Leonardo.ai
- 프롬프트 입력
- Model: Leonardo Diffusion XL
- Aspect Ratio: 21:4 또는 Custom (1920x400)
- Resolution: 1920x400

### 5. Ideogram
- 프롬프트 입력
- Style: Realistic 또는 Cinematic
- Aspect Ratio: 21:4 또는 Custom
- "No text" 옵션 선택

## 배너 이미지 권장 사양

### 크기
- **표준**: 1920x400px (4.8:1 비율)
- **넓은 배너**: 1920x300px (6.4:1 비율)
- **일반 배너**: 1920x500px (3.84:1 비율)

### 해상도
- 최소: 1920px 너비
- 권장: 2560px 너비 (고해상도 디스플레이 대응)

### 파일 형식
- PNG (투명 배경 가능)
- JPG (일반 배경)
- WebP (최적화)

## 색상 팔레트 참고
- **하늘색**: #B8D4E3 (--accent-sky)
- **피치**: #FFD4B8 (--accent-peach)
- **라벤더**: #E8D5E3 (--accent-lavender)
- **민트**: #D4F0E8 (--accent-mint)
- **옐로우**: #FFF4D4 (--accent-yellow)

## Negative Prompt (제외할 요소)
```
dark, scary, cluttered, messy, text, logo, watermark, people, faces, crowded, chaotic, cold lighting, harsh shadows, medical equipment, clinical atmosphere, words, letters, typography
```

## 배너 이미지 사용 방법

### 1. 이미지 생성
- 위 프롬프트 중 하나 선택
- AI 이미지 생성 도구 사용
- 1920x400 또는 1920x300 크기로 생성

### 2. 이미지 저장
- 파일명: `banner_1.png`, `banner_2.png` 등
- 위치: `public/assets/images/` 또는 업로드 후 URL 사용

### 3. 관리자 페이지에서 등록
1. `/blog/admin/banners` 접속
2. "새 배너 생성" 클릭
3. 이미지 업로드 또는 URL 입력
4. 제목, 내용, 기간 설정
5. 저장

### 4. 이미지만 표시하는 경우
- 이미지 업로드만 하면 됨
- 제목/내용은 선택사항
- 기본 디자인 없이 이미지만 표시됨

### 5. 텍스트와 함께 표시하는 경우
- 이미지 + 제목/내용 입력
- 이미지와 텍스트가 함께 표시됨

## 배너 이미지 개수 추천
- **기본**: 2-3개
- **이벤트/프로모션**: 필요에 따라 추가
- **롤링 표시**: 여러 배너가 자동으로 전환됨

## 디자인 팁

### 좋은 배너 이미지
✅ 부드러운 파스텔 색상
✅ 깔끔하고 미니멀한 디자인
✅ 텍스트 오버레이 공간 확보
✅ 홈페이지 분위기와 일치
✅ 전문적이면서도 따뜻한 느낌

### 피해야 할 요소
❌ 너무 복잡한 패턴
❌ 강렬한 색상
❌ 텍스트가 이미 포함된 이미지
❌ 어둡거나 무서운 분위기
❌ 사람의 얼굴이 선명하게 보이는 이미지
