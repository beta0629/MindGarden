# MindGarden 크림 베이지/코코아 디자인 시스템 v0.dev 요청 프롬프트

## 🎯 v0.dev에 복사해서 사용할 완전한 프롬프트

---

```
MindGarden 크림 베이지/코코아 디자인 시스템을 React 컴포넌트로 만들어줘.

**중요: JavaScript로만 작성해줘 (TypeScript 문법 사용 금지)**

**요구사항:**
- JavaScript 사용 (interface, type, generic types 등 TypeScript 문법 절대 금지)
- shadcn/ui 컴포넌트 활용 (Card, Button, Tabs, Progress, Badge, Input, Label, Textarea, Select, Dialog, AlertDialog, Accordion, Toggle, DropdownMenu, Tooltip, Popover, NavigationMenu, Toast, Alert, Skeleton, DataTable, Calendar, Checkbox, RadioGroup, Switch, Slider, Command, Avatar, Stepper, Timeline, Separator)
- Lucide React 아이콘 사용
- Tailwind CSS 스타일링
- 반응형 디자인

**색상 팔레트:**
- 메인: 크림 베이지(#F5F5DC), 웜 크림(#FDF5E6), 바닐라 크림(#FFF8DC), 소프트 크림(#FFFACD)
- 서브: 코코아 브라운(#8B4513), 다크 코코아(#654321), 올리브 그린(#6B7C32), 세이지 그린(#9CAF88)
- 텍스트: 다크 브라운(#3C2415), 미디엄 브라운(#8B4513), 라이트 크림(#FFFEF7)
- 배경: 그라데이션 from-[#FFFEF7] via-[#F5F5DC] to-[#FDF5E6]

**구성 요소:**

1. Hero Section (고급 디자인)
   - 제목: "MindGarden 크림 베이지/코코아 디자인 시스템" (그라데이션 텍스트 효과)
   - 부제목: "따뜻하고 고급스러운 크림 베이지와 코코아 톤으로 디자인된 상담 플랫폼 UI 컴포넌트"
   - 버튼: "시작하기" (코코아 브라운, 호버 애니메이션), "더 알아보기" (아웃라인, 글라스 효과)
   - 배경: 미묘한 radial-gradient 패턴
   - 텍스트에 페이드 인 애니메이션 (animate-in, fade-in)
   - 버튼에 호버 시 스케일 변화 (hover:scale-105)

2. 통계 대시보드 (글라스모피즘)
   - 4개 통계 카드: 컴포넌트 수, 섹션 수, 색상 수, 애니메이션 수
   - 각 카드: backdrop-blur-xl, bg-white/20, border border-white/20, shadow-2xl
   - 호버 효과: hover:scale-105, hover:-translate-y-1
   - 아이콘: Lucide React (Target, Award, Star, Zap)
   - 색상: 크림 베이지 계열

3. 타이포그래피 섹션
   - 제목 스타일: H1~H6 (크림 베이지 그라데이션)
   - 본문 텍스트: 다크 브라운 계열
   - 코드 블록: 코코아 브라운 배경
   - 인용구: 웜 크림 배경, 다크 코코아 테두리

4. 버튼 컴포넌트 (8가지 변형)
   - Primary: 코코아 브라운 배경, 크림 텍스트
   - Secondary: 크림 베이지 배경, 다크 브라운 텍스트
   - Outline: 다크 코코아 테두리, 투명 배경
   - Ghost: 투명 배경, 호버 시 크림 베이지
   - Link: 다크 코코아 텍스트, 언더라인
   - Destructive: 다크 코코아 배경
   - Success: 올리브 그린 배경
   - Warning: 바닐라 크림 배경, 다크 브라운 텍스트
   - 각 버튼에 호버 애니메이션과 포커스 상태

5. 카드 컴포넌트 (6가지 변형)
   - 기본 카드: 크림 베이지 배경, 다크 브라운 텍스트
   - 글라스 카드: backdrop-blur-xl, bg-white/20
   - 아웃라인 카드: 다크 코코아 테두리
   - 플로팅 카드: 그림자 효과, 호버 시 떠오름
   - 대시보드 카드: 통계 정보 표시
   - 프로필 카드: 아바타, 이름, 직책

6. 폼 컴포넌트 (완전한 세트)
   - Input: 크림 베이지 배경, 다크 브라운 텍스트, 코코아 브라운 포커스
   - Textarea: 동일한 스타일링
   - Select: 드롭다운 스타일
   - Checkbox: 커스텀 체크박스 디자인
   - Radio: 커스텀 라디오 버튼
   - Switch: 토글 스위치
   - Slider: 범위 슬라이더
   - DatePicker: 달력 컴포넌트
   - 각 폼 요소에 유효성 검사 상태 표시

7. 모달 컴포넌트 (4가지 타입)
   - 기본 모달: 중앙 정렬, 크림 베이지 배경
   - 확인 모달: 확인/취소 버튼
   - 폼 모달: 입력 필드 포함
   - 풀스크린 모달: 전체 화면

8. 로딩 컴포넌트 (5가지 타입)
   - 스피너: 회전 애니메이션
   - 프로그레스 바: 진행률 표시
   - 스켈레톤: 콘텐츠 로딩 중
   - 펄스: 깜빡이는 효과
   - 점프: 점프하는 점들

9. 내담자 카드 (심리 상담 특화)
   - 프로필 사진, 이름, 상담사, 다음 일정
   - 진행률 표시 (프로그레스 바)
   - 상태 배지 (진행중, 완료, 대기)
   - 액션 버튼 (연락하기, 일정 변경)
   - 호버 시 카드 확대 효과

10. 통계 차트 (5가지 타입)
    - 막대 차트: 월별 상담 수
    - 선 차트: 진행률 추이
    - 파이 차트: 상담 유형 분포
    - 도넛 차트: 성과 지표
    - 스파크라인: 간단한 트렌드

11. 네비게이션 (3가지 타입)
    - 상단 네비게이션: 로고, 메뉴, 프로필
    - 사이드바: 계층적 메뉴 구조
    - 탭 네비게이션: 섹션 전환

12. 알림 시스템 (4가지 타입)
    - 성공 알림: 올리브 그린
    - 경고 알림: 바닐라 크림
    - 오류 알림: 다크 코코아
    - 정보 알림: 세이지 그린

13. 테이블 컴포넌트 (고급 기능)
    - 정렬 가능한 헤더
    - 필터링 기능
    - 페이지네이션
    - 행 선택
    - 액션 드롭다운

14. 캘린더 컴포넌트
    - 월별/주별/일별 보기
    - 이벤트 표시
    - 날짜 선택
    - 일정 추가/수정

15. 아코디언 컴포넌트
    - FAQ 스타일
    - 단일/다중 열기
    - 아이콘 애니메이션
    - 커스텀 스타일링

16. 색상 팔레트 섹션
    - 메인 색상: 크림 베이지, 웜 크림, 바닐라 크림, 소프트 크림
    - 서브 색상: 코코아 브라운, 다크 코코아, 올리브 그린, 세이지 그린
    - 각 색상에 호버 효과와 복사 기능
    - 색상 코드 표시

**프리미엄 글라스모피즘 디자인:**
- backdrop-blur-xl 효과
- bg-white/20 반투명 배경
- border border-white/20 미묘한 테두리
- shadow-2xl 깊은 그림자
- hover:shadow-inner 내부 그림자 효과

**애니메이션 효과:**
- fade-in: 페이드 인 애니메이션
- slide-in-from-bottom: 아래에서 슬라이드
- slide-in-from-right: 오른쪽에서 슬라이드
- hover:scale-105: 호버 시 확대
- hover:-translate-y-1: 호버 시 위로 이동
- pulse: 맥박 효과
- bounce: 바운스 효과

**디테일한 디자인 요소:**
- 그라데이션 텍스트: bg-gradient-to-r from-[#8B4513] to-[#6B7C32]
- 그라데이션 배경: from-[#FFFEF7] via-[#F5F5DC] to-[#FDF5E6]
- radial-gradient 패턴: circle_at_30%_50%,rgba(139,69,19,0.1)
- 골든 비율 기반 간격: 1.618 비율
- 미묘한 그림자: shadow-lg, shadow-xl, shadow-2xl
- 부드러운 모서리: rounded-lg, rounded-xl, rounded-2xl

**프리미엄 반응형 디자인:**
- 모바일: 320px~768px
- 태블릿: 768px~1024px
- 데스크톱: 1024px+
- 각 브레이크포인트별 최적화된 레이아웃
- 터치 친화적 인터페이스

**모바일 특화 디자인:**
- 스와이프 제스처 지원
- 터치 타겟 최소 44px
- 모바일 네비게이션 메뉴
- 햄버거 메뉴 애니메이션
- 풀스크린 모달

**내담자 카드 특화 디자인:**
- 상담 진행률 시각화
- 감정 상태 표시 (이모지/아이콘)
- 다음 상담까지 남은 시간
- 긴급도 표시 (색상 코딩)
- 상담사 연락처 바로가기

**사용 방법:**
1. v0.dev에 이 프롬프트를 복사해서 붙여넣기
2. "JavaScript로만 만들어줘"라고 추가 요청
3. 생성된 코드를 React 프로젝트에 복사
4. 필요한 패키지 설치 (shadcn/ui, lucide-react)

**포함된 모든 컴포넌트:**
- Hero Section, 통계 대시보드, 타이포그래피
- 버튼 (8종), 카드 (6종), 폼 (8종), 모달 (4종)
- 로딩 (5종), 내담자 카드, 차트 (5종)
- 네비게이션 (3종), 알림 (4종), 테이블
- 캘린더, 아코디언, 색상 팔레트
- 총 16개 섹션, 50+ 컴포넌트

**최종 요청:**
위의 모든 요구사항을 포함한 완전한 MindGarden 크림 베이지/코코아 디자인 시스템을 JavaScript React 컴포넌트로 만들어줘. TypeScript 문법은 절대 사용하지 말고, 모든 색상은 정확한 hex 코드를 사용해줘.
```
