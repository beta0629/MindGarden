# 비디오 파일 안내

이 폴더에 상담소 관련 비디오 파일을 넣어주세요.

## 권장 사양
- 형식: MP4 (H.264 코덱)
- 해상도: 1920x1080 (Full HD) 이상
- 파일명: `hero-video.mp4`

## 비디오 내용 예시
- 상담실 내부 모습
- 따뜻하고 편안한 상담 공간
- 내담자들이 웃고 있는 모습 (프라이버시 고려)
- 상담소의 평화로운 분위기

## 사용 방법
1. 비디오 파일을 이 폴더에 `hero-video.mp4`로 저장
2. 자동으로 히어로 섹션에 표시됩니다
3. 코어솔루션 API에서 비디오 URL을 제공하면 API 우선 사용
