# 상담사·내담자 핵심 화면 와이어프레임 & 스펙

본 문서는 앱의 레이아웃, 컴포넌트 배치, 정보 계층을 설명하는 텍스트 기반 와이어프레임입니다.
각 화면은 `AppShell`과 `BottomNavigation`을 기본으로 감싸고 있습니다.

## 1. 상담사 (Consultant) - 대시보드 (홈)

```text
[Top Bar]
---------------------------------------------------
|  (MindGarden 로고)           [🔔] [프로필 아바타]  |
---------------------------------------------------

[Main Content - Scrollable]
---------------------------------------------------
| 안녕하세요, 김상담 선생님!                          |
| 오늘 3건의 상담이 예정되어 있습니다.                  |
---------------------------------------------------
| [긴급 알림 영역] (Color: Error)                     |
| ⚠️ 미작성 일지 1건이 있습니다. [바로가기 >]           |
---------------------------------------------------
| [Section: 오늘의 스케줄]                            |
| ----------------------------------------------- |
| | 10:00 - 10:50                               | |
| | 이내담 님 (초기상담)                            | |
| | [상담 일지 열기] [입장 대기]                    | |
| ----------------------------------------------- |
| ----------------------------------------------- |
| | 14:00 - 14:50                               | |
| | 박내담 님 (정기상담)                            | |
| | [일정 변경] [입장 대기]                        | |
| ----------------------------------------------- |
---------------------------------------------------
| [Section: 빠른 액션]                               |
| [일정 추가 ➕]   [근무 설정 ⏰]   [수입 내역 💰]      |
---------------------------------------------------

[Bottom Navigation]
---------------------------------------------------
|  [🏠]      [📅]      [👥]      [📝]      [⚙️]   |
|   홈      스케줄     내담자      일지     더보기    |
---------------------------------------------------
```

## 2. 상담사 (Consultant) - 상담 일지 작성

```text
[Top Bar]
---------------------------------------------------
| [< 뒤로]    일지 작성                     [저장]    |
---------------------------------------------------

[Main Content]
---------------------------------------------------
| 내담자: 이내담 님 | 2026.05.12 10:00 - 10:50        |
---------------------------------------------------
| [Section: 상담 요약 (공유됨)]                       |
| 내담자에게 공유되는 한 줄 요약입니다.                   |
| [ 텍스트 입력 영역 (Input) ................ ]       |
---------------------------------------------------
| [Section: 전문가 메모 (비공개)]                     |
| 상담사만 볼 수 있는 상세 메모입니다.                    |
| [ 텍스트 영역 (Textarea, min-height: 200px) ]    |
---------------------------------------------------
| [Section: 태그 추가]                                |
| (칩) 우울  (칩) 불안  (칩) 가족문제  [+ 추가]          |
---------------------------------------------------
| [Section: 다음 상담 제안]                           |
| [날짜 선택기] [시간 선택기]                           |
---------------------------------------------------
```

## 3. 내담자 (Client) - 피드 (홈)

```text
[Top Bar]
---------------------------------------------------
|  (MindGarden 로고)           [🔔] [프로필 아바타]  |
---------------------------------------------------

[Main Content]
---------------------------------------------------
| 이내담 님, 환안한 하루 되세요.                       |
---------------------------------------------------
| [Section: 다가오는 상담 - ConsultationCard]          |
| ----------------------------------------------- |
| | [배지: D-1]                                 | |
| | [상담사 썸네일] 김상담 선생님                     | |
| | 5.13(수) 14:00 - 14:50                      | |
| | [상담실 입장하기 (비활성)]                      | |
| ----------------------------------------------- |
---------------------------------------------------
| [Section: 빠른 예약]                               |
| [새로운 상담 예약하기 (Primary Button, 100% width)]  |
---------------------------------------------------
| [Section: 오늘의 웰니스 - WellnessCard]              |
| ----------------------------------------------- |
| | 🎧 3분 호흡 명상                              | |
| | 마음을 차분하게 가라앉히는 짧은 명상입니다.         | |
| ----------------------------------------------- |
---------------------------------------------------

[Bottom Navigation]
---------------------------------------------------
|  [🏠]      [📅]      [💬]      [🌿]      [⚙️]   |
|   홈       예약     내 상담    웰니스     더보기    |
---------------------------------------------------
```

## 4. 내담자 (Client) - 예약 플로우 (스텝 통합)

```text
[Top Bar]
---------------------------------------------------
| [< 뒤로]    새로운 상담 예약                       |
---------------------------------------------------
| [Progress Bar: 33% -----------------------]     |
---------------------------------------------------

[Main Content]
---------------------------------------------------
| 1단계: 상담사 선택                                  |
| 검색어 입력 [ 🔍 상담사 이름, 전문분야 ]               |
---------------------------------------------------
| [상담사 리스트 카드형]                              |
| ----------------------------------------------- |
| | [썸네일] 김상담 전문가                          | |
| | ★ 4.9 (120개 리뷰)                           | |
| | 전문분야: 우울, 대인관계, 자존감                   | |
| ----------------------------------------------- |
| ----------------------------------------------- |
| | [썸네일] 박상담 전문가                          | |
| | ★ 4.8 (85개 리뷰)                            | |
| | 전문분야: 진로, 스트레스 관리                      | |
| ----------------------------------------------- |
---------------------------------------------------

(상담사 선택 후 2단계 뷰로 전환)
---------------------------------------------------
| 2단계: 시간 선택 (김상담 전문가)                      |
| [ 🗓️ 2026년 5월 ] (주간 캘린더 스와이프)             |
| [ 월 11 ] [ 화 12 ] [ 수 13 ] [ 목 14 ] [ 금 15 ]  |
|                                                 |
| 가용 시간:                                        |
| (칩) 10:00  (칩 활성) 14:00  (칩) 16:00            |
---------------------------------------------------

(하단 고정 버튼 영역 - Sticky)
---------------------------------------------------
| [다음 단계로 (결제)]                                |
---------------------------------------------------
```
