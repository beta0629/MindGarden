# 관리자 가이드

마인드가든 홈페이지 관리자 로그인 및 사용 가이드입니다.

## 🔐 관리자 로그인

### 1. 로그인 페이지 접근

**방법 1: 직접 접근**
```
http://localhost:3000/blog/admin/login
```
또는 배포된 도메인:
```
https://your-domain.com/blog/admin/login
```

**방법 2: 관리 페이지 접근 시 자동 리다이렉트**
- `/blog/admin` 또는 `/blog/admin/list` 접근 시
- 로그인되지 않은 경우 자동으로 로그인 페이지로 이동

### 2. 비밀번호 입력

로그인 페이지에서 관리자 비밀번호를 입력합니다.

**비밀번호 설정 위치:**
- 환경 변수 파일: `.env.local`
- 변수명: `BLOG_ADMIN_PASSWORD`

**예시:**
```env
BLOG_ADMIN_PASSWORD=MySecurePassword123!
```

**기본값:**
- 환경 변수가 설정되지 않은 경우: `admin123`
- ⚠️ **프로덕션에서는 반드시 변경하세요!**

### 3. 로그인 상태

- 로그인 성공 시 쿠키에 인증 토큰이 저장됩니다
- 로그인 상태는 **30일간 유지**됩니다
- 브라우저를 닫아도 로그인 상태가 유지됩니다

## 📝 블로그 관리 기능

### 글 작성

1. 로그인 후 `/blog/admin` 접근
2. 다음 정보 입력:
   - **제목** (필수)
   - **요약** (선택)
   - **내용** (필수, HTML 사용 가능)
   - **썸네일 이미지** (선택, 업로드 가능)
   - **상태**: 임시 저장 / 발행
   - **홈페이지 전용**: 체크 시 홈페이지에서만 표시

3. "저장하기" 버튼 클릭

### 글 목록 관리

1. `/blog/admin/list` 접근
2. 모든 글 목록 확인:
   - 발행됨 / 임시저장 상태 표시
   - 홈페이지 전용 표시
   - 작성/발행 날짜 표시

3. 각 글에 대해:
   - **보기**: 블로그 상세 페이지로 이동
   - **수정**: 글 수정 페이지로 이동
   - **삭제**: 글 삭제 (확인 후 삭제)

### 글 수정

1. 목록에서 "수정" 버튼 클릭
2. 또는 상세 페이지의 관리자 메뉴에서 "수정" 클릭
3. 내용 수정 후 "수정하기" 버튼 클릭

### 글 삭제

1. 목록에서 "삭제" 버튼 클릭
2. 확인 메시지에서 확인
3. 삭제된 글은 복구할 수 없습니다

## 🔒 보안 권장사항

1. **강력한 비밀번호 사용**
   - 최소 12자 이상
   - 대소문자, 숫자, 특수문자 조합
   - 예: `MindGarden2025!@#`

2. **환경 변수 보호**
   - `.env.local` 파일은 Git에 커밋하지 않음
   - 프로덕션 환경에서는 서버 환경 변수로 설정

3. **정기적인 비밀번호 변경**
   - 보안을 위해 주기적으로 비밀번호 변경 권장

4. **로그아웃**
   - 공용 컴퓨터 사용 시 반드시 로그아웃
   - 관리 페이지 우측 상단의 "로그아웃" 버튼 클릭

## 🆘 문제 해결

### 로그인이 안 되는 경우

1. **비밀번호 확인**
   - 환경 변수 `BLOG_ADMIN_PASSWORD`가 올바르게 설정되었는지 확인
   - 서버 재시작 후 다시 시도

2. **쿠키 확인**
   - 브라우저 쿠키가 차단되어 있는지 확인
   - 시크릿 모드에서 시도

3. **서버 로그 확인**
   - 서버 콘솔에서 오류 메시지 확인

### 로그인 상태가 유지되지 않는 경우

1. 브라우저 쿠키 설정 확인
2. 쿠키가 삭제되었는지 확인
3. 다시 로그인 시도
