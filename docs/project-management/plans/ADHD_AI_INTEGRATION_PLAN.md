# ADHD 특화 프로그램 AI 도입 계획서

**작성일**: 2025-12-28  
**버전**: 1.0.0  
**상태**: 계획 수립  
**작성자**: CoreSolution Development Team

---

## 📋 개요

CoreSolution 플랫폼에 ADHD(주의력결핍 과잉행동장애) 특화 프로그램에서 AI 기술을 도입하여 진단, 치료, 관리의 효율성을 향상시키는 계획서입니다.

### 배경

- ADHD 환자의 수요가 지속적으로 증가하고 있으나, 전문적인 관리 도구가 부족함
- 기존 상담 방식의 한계로 인한 객관적 데이터 부족
- AI 기술을 활용한 개인 맞춤형 관리 프로그램의 필요성
- 아직 완전히 상용화되지 않은 분야로 시장 기회 존재

### 목표

1. **진단 및 선별 도구 개발**: 객관적이고 신속한 ADHD 진단 보조
2. **일상 모니터링 시스템 구축**: 실시간 행동 분석 및 상태 추적
3. **맞춤형 디지털 치료 프로그램 제공**: 개인별 특성에 맞는 치료 방안 제시
4. **예방 및 조기 개입 시스템**: 증상 악화 위험 조기 감지 및 상담사 알림

## 🎯 AI 도입 필요 영역

### 1. 진단 및 선별 도구

#### 현재 상태
- 연구 단계에 있으나 상용화는 초기 단계
- 세브란스병원: 망막 안저 사진 분석으로 96.9% 정확도 달성
- 뇌파(EEG) 신호 분석을 통한 진단 보조 연구 진행 중

#### ⚠️ 사용 환경 제약사항

**망막 안저 사진 분석**
- **의료기기 필요**: 특수 망막 안저 촬영 카메라 (Fundus Camera)
- **의료진 필요**: 안과 전문의 또는 검사사
- **사용 장소**: 병원, 의료기관 (상담센터 직접 사용 어려움)
- **의료기기 인증**: FDA/KFDA 승인 필요
- **비용**: 고가의 의료기기 및 검사 비용

**결론**: 상담센터에서 직접 사용하기보다는 **의료기관과의 협력** 또는 **의료기관 전용 솔루션**으로 개발하는 것이 적절함

#### 원리 및 작동 방식

**망막과 뇌의 연결**
- 망막은 뇌의 연장선으로, 신경 조직으로 구성됨
- 망막의 구조적 변화가 뇌의 신경 발달 상태를 반영
- 뇌의 기능 이상이 망막에 구조적 변화로 나타남

**분석 대상 특징**
1. **혈관 밀도 변화**
   - ADHD 환자에서 혈관 밀도가 증가하는 경향
2. **동맥 혈관 폭 감소**
   - 정상 대비 동맥 혈관이 좁아지는 현상
3. **시신경유두 구조 변화**
   - 시신경의 구조적 이상이 발견됨
4. **신경 섬유층 변화**
   - 망막 신경 섬유층의 두께 및 구조 변화

**AI 분석 과정**
1. 망막 안저 사진 촬영 (특수 카메라)
2. 딥러닝 모델이 이미지에서 특징 추출
3. 학습된 패턴과 비교하여 ADHD 확률 계산
4. 96.9% 정확도로 ADHD/정상 분류

**장점**
- 비침습적 (검사가 간단하고 안전함)
- 신속한 검사 (기존 진단 방식보다 빠름)
- 객관적 평가 (주관적 설문 보완)
- 비용 효율적

#### CoreSolution 통합 방안

**옵션 A: 역할 분리 모델** (보조 수단으로 활용 가능) ⭐ 권장
- **촬영**: 의료기관에서 수행 (병원, 안과 클리닉)
  - 환자가 병원에서 망막 안저 사진 촬영
  - 촬영된 이미지 파일 제공 받음
- **분석**: 상담센터에서 CoreSolution AI 분석
  - 촬영된 망막 안저 이미지를 CoreSolution에 업로드
  - AI 모델이 이미지를 분석하여 ADHD 선별 결과 제공
  - 상담사가 결과를 확인하고 상담에 활용
- **장점**:
  - 상담센터에서 의료기기 구비 불필요 (이미지만 있으면 됨)
  - 보조 수단으로 객관적 데이터 활용 가능
  - 상담 효과 측정 및 검증 도구로 활용
  - 환자가 병원 결과를 가져오기만 하면 됨
- **단점**:
  - 환자가 별도로 병원 방문 필요
  - 이미지 파일 형식 표준화 필요
- **사용 시나리오**:
  1. 환자가 병원에서 망막 안저 촬영 (1회 또는 정기)
  2. 이미지 파일을 CoreSolution에 업로드
  3. AI 분석 결과 확인 (ADHD 위험도 점수, 특징 설명)
  4. 상담사가 결과를 참고하여 상담 계획 수립
  5. 치료 진행 중 정기적으로 재검사하여 변화 추적

**옵션 B: 의료기관 협력 통합 모델**
- **의료기관에서 검사 수행**: 병원에서 망막 안저 촬영
- **CoreSolution API 연동**: 검사 결과를 자동으로 CoreSolution으로 전송
- **상담센터 활용**: 실시간으로 검사 결과 확인 및 상담 진행
- **장점**: 자동화된 데이터 연동
- **단점**: 의료기관과 API 통합 협의 필요

**옵션 C: 상담센터용 대체 방법** (독립적 사용)
- **망막 안저 대신 다른 방법 사용**: 상담센터에서 직접 사용 가능한 선별 도구
- **예시**: 
  - 행동 관찰 기반 AI 분석
  - 설문 + 행동 데이터 종합 분석
  - 뇌파(EEG) 간단한 측정 (휴대용 장비)
- **장점**: 의료기관 의존성 없이 독립적 사용

#### 기술 요구사항
- **컴퓨터 비전 AI**: 망막 안저 사진 이미지 분석
  - Convolutional Neural Network (CNN)
  - Transfer Learning (의료 이미지 사전 학습 모델 활용)
- **이미지 전처리**: 노이즈 제거, 정규화
- **분류 모델**: ADHD/정상 이진 분류 또는 위험도 점수 산출
- **의료기기 연동**: 망막 안저 촬영 장비와의 API 연동

#### 우선순위
- **중장기** (의료 기관 협력 및 규제 승인 필요)
- **상담센터 직접 사용 어려움**: 의료기관 협력 또는 별도 솔루션 필요
- **대안**: 상담센터에서는 다른 선별 방법(행동 분석, 설문 등) 우선 고려

### 2. 일상 모니터링 및 행동 분석

#### 현재 상태
- 일부 앱 존재하나 완전 자동화는 부족
- 실시간 분석 및 상담 연계 기능 부족

#### CoreSolution 통합 방안
- **실시간 행동 패턴 모니터링**: 일상 생활 중 집중력, 활동량 추적
- **생체 신호 분석**: 웨어러블 기기 연동 (심박수, 활동량 등)
- **상담 데이터 연계**: 상담사에게 객관적 데이터 제공
- **이상 징후 감지**: 이상 패턴 발견 시 자동 알림

#### 기술 요구사항
- **행동 패턴 분석 AI**
  - 카메라 기반 행동 분석 (선택적)
  - 앱 사용 패턴 분석
  - 키보드 타이핑 패턴 분석
- **생체 신호 분석 AI**
  - 웨어러블 기기 데이터 처리
  - 심박 변이도(HRV) 분석
  - 수면 패턴 분석
- **시계열 데이터 분석**: 장기적 패턴 추적 및 변화 감지

#### 데이터 수집 방법
1. **스마트폰 앱**
   - 화면 사용 시간
   - 앱 전환 빈도
   - 타이핑 속도 및 패턴
2. **웨어러블 기기** (선택적)
   - 심박수
   - 활동량
   - 수면 데이터
3. **일일 설문** (간단한 체크리스트)
   - 집중도
   - 기분 상태
   - 약물 복용 여부

#### 우선순위
- **즉시 도입 가능** (높은 가치)
- CoreSolution 기존 시스템과 연계 용이
- 상담 효과 측정 및 개선에 직접 활용

### 3. 맞춤형 디지털 치료 프로그램

#### 현재 상태
- EndeavorRX (FDA 승인, 8-12세 아동용) 존재
- VR 기반 바디 더블링(body doubling) 연구 진행 중
- 성인 ADHD 및 맞춤형 프로그램은 부족

#### CoreSolution 통합 방안
- **상담사 처방형 프로그램**: 상담사가 환자 상태에 맞게 프로그램 처방
- **개인별 최적화**: AI가 개인 특성 분석하여 프로그램 조정
- **게임화된 치료**: 집중력 향상 게임, 인지 훈련 프로그램
- **VR 기반 치료** (선택적): 집중력 향상을 위한 가상 환경 제공

#### 주요 기능
1. **집중력 훈련 게임**
   - 개인별 난이도 자동 조정
   - 진행 상황 추적
   - 약한 영역 집중 훈련
2. **인지 훈련 프로그램**
   - 작업 기억력 향상
   - 주의력 분산 훈련
   - 계획 수립 및 실행 훈련
3. **생활 습관 개선 프로그램**
   - 일정 관리 도구
   - 작업 분할 기법
   - 시간 관리 스킬

#### 기술 요구사항
- **강화학습 AI**: 개인별 최적 난이도 자동 조정
- **추천 시스템**: 개인 특성 기반 프로그램 추천
- **게임 엔진 통합**: Unity 또는 웹 기반 게임
- **VR 통합** (선택적): VR 환경 제공

#### 우선순위
- **즉시 도입 가능** (차별화 포인트)
- CoreSolution의 기존 상담 서비스 확장
- 상담사와 환자 모두에게 가치 제공

## 🚀 CoreSolution 통합 우선순위

### Phase 1: 즉시 도입 가능 (높은 가치) - 3-6개월

#### 1.1 일상 모니터링 및 행동 분석 시스템
- **목표**: ADHD 환자의 일상 행동 패턴 실시간 추적
- **기능**:
  - 앱 사용 패턴 분석
  - 생체 신호 수집 (웨어러블 기기 연동)
  - 일일 체크리스트 수집
  - 대시보드 제공 (환자, 상담사)
- **통합 포인트**:
  - CoreSolution 상담 시스템과 연계
  - 상담 전/후 데이터 비교
  - 상담사 리포트 자동 생성

#### 1.2 맞춤형 디지털 치료 프로그램
- **목표**: 상담사 처방형 개인 맞춤 치료 프로그램
- **기능**:
  - 집중력 훈련 게임
  - 인지 훈련 프로그램
  - 생활 습관 개선 도구
  - 진행 상황 추적
- **통합 포인트**:
  - CoreSolution 상담 서비스의 일부로 제공
  - 상담사가 프로그램 처방 및 모니터링
  - 환자 진행 상황을 상담 데이터로 활용

#### 1.3 실시간 집중력 추적
- **목표**: 상담 세션 및 일상 활동 중 집중도 측정
- **기능**:
  - 화상 상담 중 집중도 측정 (선택적)
  - 앱 사용 중 집중 패턴 분석
  - 주의 산만 감지 및 알림
- **통합 포인트**:
  - 상담 효과 측정 도구
  - 상담사에게 실시간 피드백 제공

### Phase 2: 중기 도입 (연구 및 개발 필요) - 6-12개월

#### 2.1 학습/업무 환경 최적화 AI
- **목표**: 개인별 최적 학습/업무 환경 제안
- **전제 조건**: Phase 1의 모니터링 데이터 축적 필요

#### 2.2 예측 및 예방 시스템
- **목표**: 증상 악화 위험 조기 감지
- **전제 조건**: 장기 데이터 축적 필요

### Phase 3: 장기 도입 (보조 수단으로 활용) - 12개월 이상

#### 3.1 진단 및 선별 도구 (보조 수단)
- **목표**: 객관적 ADHD 진단 보조 (상담센터에서 이미지 분석)
- **역할 분리**:
  - **촬영**: 의료기관 (병원, 안과 클리닉)
  - **분석**: CoreSolution AI (상담센터에서 이미지 업로드 후 분석)
  - **활용**: 상담사가 결과를 참고하여 상담 진행
- **전제 조건**:
  - AI 모델 개발 (망막 안저 이미지 분석)
  - 이미지 업로드 및 분석 기능 개발
  - 분석 결과 리포트 생성 기능
  - 의료기관과의 협력 (선택적, 환자가 직접 병원 방문도 가능)
- **규제 고려사항**:
  - **진단 목적이 아닌 보조 수단**: 진단은 의료진이 하며, AI는 보조 정보 제공
  - 의료기기 인증 불필요 (분석 소프트웨어로 분류 가능)
  - 단, 법적 검토 필요 (의료법 준수)
