# Core Solution Identity (SSOT)

> **Status**: Active  
> **Scope**: Core Solution 제품 정체성·톤·Calm Forest 시각 방향. 메인 앱 UI Phase 1 시안 입력.

## 1. Core Identity
- **한 줄 정의**: 상담·심리·웰니스 센터의 복잡한 현장 업무를 하나로 연결하는 통합 운영 플랫폼
- **타깃**: 센터 원장 및 실무 담당자 (상담사, 행정 직원)
- **약속**: 예약, 회기 관리, 정산(ERP)을 한곳에서 끊김 없이
- **톤 (Tone & Manner)**: Calm (차분함), Trust (신뢰), Field-oriented (현장 중심)

## 2. Brand Positioning

### Core Solution vs Trinity
- **Trinity**: 회사(Corporate) 브랜드. 투자자, 파트너, 법인(B2B) 대상의 소개 및 영업 채널, 회사 랜딩/온보딩에 사용됩니다.
- **Core Solution**: 제품(Product) 브랜드. 마인드가든(MindGarden) 상담 센터 운영 플랫폼 자체를 의미하며, 실제 사용자인 센터 원장·실무진 대상의 제품 공개 랜딩, 앱, 마케팅에 사용됩니다. 제품 랜딩에서 Trinity 회사명이나 CI 혼용을 엄격히 지양하여 제품 자체의 독립적 정체성을 확립합니다.

### Core Solution vs Generic SaaS
- **Generic SaaS**: 일반적인 근태 관리, 범용 캘린더, 단순 회계 등 산업 특화가 부족하여 현장 업무를 온전히 담아내지 못하는 파편화된 IT 도구.
- **Core Solution**: 현장(Field)의 실제 워크플로에 깊이 밀착된 버티컬 솔루션. 단순 일정을 넘어 '내담자-상담사 매핑', '회기(Session) 진행 상태', 'ERP 기반 정산' 등 상담/웰니스 센터의 핵심 비즈니스 파이프라인을 완벽히 지원합니다. (향후 다업종 현장 운영 플랫폼으로의 확장성 내포)

## 4. Visual Direction

### 4.1. Imagery (이미지 활용)
- **실제 운영 화면 (UI Mockups)**: 대시보드, 통합 스케줄러, 회기 관리 등 실제 사용자가 매일 마주하는 직관적이고 깔끔한 UI 화면을 적극 노출하여 실체감과 신뢰도를 높입니다.
- **현장 중심 (Field-oriented)**: 차분한 분위기의 상담 센터 내부, 편안하게 대화하는 상담사와 내담자, 여유롭게 업무를 보는 실무진의 모습 등 실제 현장의 따뜻하고 정돈된 이미지를 활용합니다.

### 4.2. Color Emotion (색상 감성)
- **Calm Forest Palette**: 차분하고 안정감을 주는 그린 계열(`mg-v2-color-primary`, `#3D5246`)을 주조색으로 사용하여 신뢰감과 편안함을 전달합니다.
- **Surface & Background**: 부드러운 오프화이트/베이지 톤(`surface`, `#FAF9F7`)을 배경으로 사용하여 눈의 피로를 덜고 따뜻한 감성을 부여합니다.

### 4.3. Illustration Style
- **미니멀 & 플랫 (Minimal & Flat)**: 복잡한 3D나 화려한 그래픽보다는, 선과 면을 활용한 깔끔하고 정돈된 플랫 일러스트레이션을 지향합니다.
- **기능적 은유 (Functional Metaphor)**: 흩어진 퍼즐이 맞춰지거나, 복잡한 선이 하나로 정리되는 등 '통합'과 '효율'을 은유적으로 표현하는 그래픽 요소를 활용하여 IT 솔루션의 딱딱함을 중화시킵니다.
