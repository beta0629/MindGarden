# 통합 로그인 페이지 전면 리뉴얼 기획

**작성일**: 2026-03-04
**역할**: core-planner (기획 및 실행 분배)
**대상**: `frontend/src/components/auth/UnifiedLogin.js` 및 관련 CSS (`UnifiedLogin.css` 등)

---

## 1. 목표 및 배경
- **요구사항**: 통합 로그인 페이지를 완전히 새롭게 리뉴얼하고, 사진/이미지를 포함한 현대적이고 세련된 아토믹 디자인 패턴으로 변경한다.
- **제약사항**: 카카오(Kakao)와 네이버(Naver) 소셜 로그인 기능은 기존대로 유지해야 한다.
- **방향성**: 현재의 단순한 폼 형태에서 벗어나, 좌측 또는 우측에 풀사이즈 브랜딩 이미지(사진)를 배치하고 한쪽에 로그인 폼을 배치하는 분할 레이아웃(Split Layout) 등을 적용하여 시각적 완성도를 높인다.

---

## 2. 범위
- **포함**: `UnifiedLogin.js` 컴포넌트 마크업 구조 변경, `UnifiedLogin.css` 스타일 리뉴얼, 로그인 폼/버튼/소셜 로그인 버튼 아토믹 컴포넌트 적용.
- **유지**: 로그인 로직(API 호출, 세션 확인, 멀티 테넌트 처리 등), 카카오/네이버 로그인 연동 처리 함수 (`kakaoLogin`, `naverLogin`).

---

## 4. 디자인 및 레이아웃 제안 (디자이너 참고용)
- **레이아웃**: 데스크탑에서는 화면을 5:5 또는 4:6으로 분할하여 한쪽에는 MindGarden 브랜딩 사진(고품질 이미지 또는 그라데이션+이미지)을 배치하고, 다른 한쪽 중앙에 로그인 폼을 배치한다. 모바일에서는 사진 영역을 숨기거나 상단 배너 형태로 축소한다.
- **사진 애셋**: 임의의 placeholder 이미지(예: Unsplash 등)를 사용하거나 프로젝트 내 적절한 이미지를 활용.
- **아토믹 구성**: 
  - Atom: `mg-v2-button-primary`, `mg-v2-button-outline` (소셜), `mg-v2-input`
  - Molecule: 이메일/비밀번호 Form 필드, 소셜 로그인 버튼 그룹
  - Organism: 로그인 폼 영역 전체, 사진 영역
  - Template: Split Screen Layout
- **소셜 로그인**: 카카오, 네이버 버튼은 브랜드 고유 색상 또는 통일된 아토믹 버튼(로고 아이콘 포함) 형태로 세련되게 배치.
