# 상담의 종류 페이지 이미지 보강 기획

**문서 역할**: `/counseling/*` 4정적 페이지에 이미지를 넣기 위한 기획 스펙.  
**구현 순서**: 디자이너 → 퍼블리셔 → 코더.  
**관련 코드**: `lib/counseling-type-pages.ts`, `components/CounselingTypePageView.tsx`, `app/globals.css` (`.counseling-type-*`).

---

## 1. 목표

- **신뢰**: 센터 톤(차분함, 전문성, 안전한 공간)을 시각적으로 뒷받침하고, “교육용 텍스트만 있는 페이지” 느낌을 줄인다.
- **이해 보조**: 연령·맥락(가정·학교, 직장·일상, 동반 어려움의 복합성 등)을 **장면·분위기** 단서로 보조한다. 평가·효과를 **그림으로 암시하지 않는다**.
- **센터 톤**: 홈 `home-section-images`와 같이 **따뜻하고 과하지 않은 실내·공간·자연광** 계열로 브랜드 일관성을 맞춘다.
- **스캔 가능성**: 긴 카드 스택 사이에 시각적 리듬을 넣어 스크롤 피로를 완화한다.

---

## 2. 이미지 정책 정합

| 출처 | 내용 |
|------|------|
| **공개 UI** | 유니코드 이모지 금지. 분위기 보조는 **실제 이미지**(또는 SVG 아이콘)로 처리. |
| **GNB·문의** | 문의·상담 예약은 **바텀시트 전용**. 이미지 안에 “문의하기” 버튼·전화번호 오버레이·`#contact` 유도 등 **GNB/문의 중복 CTA를 넣지 않는다**. |
| **콘텐츠** (`lib/counseling-type-pages.ts`) | 효과·평가 보장 문구 없음. 이미지도 **전후 비교, 특정 결과, 의료 행위 암시(처방·수술 등)** 를 담지 않는다. |
| **접근성·alt** | h2/제목 문장을 alt에 그대로 반복하지 않고 **장면·분위기**를 한 문장으로 설명. 순수 장식이면 팀 합의 하에 장식 처리 검토. |
| **실사 vs 일러스트** | *(가정)* 홈과 동일하게 **실사(센터·휴식 공간 계열) 우선**. 사람이 나오면 **모델·촬영 동의·얼굴 노출 정책**을 디자인 단계에서 확정. |
| **자산 위치** | *(가정)* 브랜드·법적 통제를 위해 **`public/` 정적 자산 우선** 권장. |

## 3. 페이지별·섹션별 제안

**공통**: **히어로(h1 + lead) 직후**에 페이지 대표 이미지 1장. 카드 안 vs 카드 사이 레이아웃은 디자인·퍼블에서 결정하되, IA상 **위치만** 아래 표를 따른다.

### `/counseling/child-adolescent-adhd`

| 우선순위 | 위치 | 비고 |
|----------|------|------|
| 1 | 히어로 직후 (`who` 카드 전) | 가족·성장·안전한 공간 톤. 특정 아동 식별 불가한 장면 권장. |
| 2 | `features` 앞 또는 카드 상단 | 학교·루틴 맥락을 은유적으로. 과잉행동 연출 금지. |
| 3 (선택) | `family` 직전 또는 직후 | 보호자 협력 문단과 톤 정합. |

**권장 장수**: 2~3장. `related`, `process` 전용 대형 이미지는 비권장.

### `/counseling/adult-adhd`

| 우선순위 | 위치 | 비고 |
|----------|------|------|
| 1 | 히어로 직후 | 성인 일상·직장 맥락을 일반화된 장면으로. |
| 2 | `presentation` 또는 `focus` 앞 | “맥락에서의 기능” 이해 보조. |
| 3 (선택) | `comorbid` 앞 | 정서적 부담을 판타지/공포 이미지 없이 차분하게. |

**권장 장수**: 2~3장. `related` 직전 대형 배너 비권장.

### `/counseling/comorbidity`

| 우선순위 | 위치 | 비고 |
|----------|------|------|
| 1 | 히어로 직후 | 통합적·균형 잡힌 분위기. 의학 도식 과용 지양. |
| 2 | `areas` 앞 | 동반 영역 나열 전 시각적 완충. |
| 3 (선택) | `counseling` 앞 | 상담 접근 문단 진입부 리듬. |

**주의**: `medical` 섹션은 **의료 행위 시각화**와 혼동될 수 있어 이미지 **바로 옆** 배치는 피한다.

**권장 장수**: 2~3장.

### `/counseling/counseling-areas`

| 우선순위 | 위치 | 비고 |
|----------|------|------|
| 1 | 히어로 직후 | 다루는 방향과 맞는 중립적 전문 공간 톤. |
| 2 | `examples` 앞 | 과목 예시의 다양성을 과장 없이. |
| 3 (선택) | `intake` 앞 | 초기 상담의 신뢰·절차감(대기실·상담실 분위기 수준). |

**비권장**: `out-of-scope`에 부정적·거절 연상 이미지.

**권장 장수**: 2~3장.

### 요약

- **최소**: 히어로 1장만.
- **권장**: 페이지당 2~3장.
- **상한**: 4장 (모바일 스크롤·LCP 부담).

## 4. 에셋 스펙

| 항목 | 제안 |
|------|------|
| 비율 | 히어로·구간: **16:9** 또는 **3:2**. 카드 내 썸네일: **4:3**. |
| 해상도 | 표시 폭 기준 **약 1200~1600px** (레티나 2x 대비 원본은 그에 맞게). |
| 포맷 | **WebP** 우선, 폴백 JPEG. |
| 파일 경로 컨벤션 | `public/assets/images/counseling/{slug}/hero.webp`, `section-{sectionId}.webp` 등. |
| alt | (1) 페이지 제목 복붙 금지. (2) 보이는 것 + 분위기 한 문장. (3) 평가명·효과 금지. |
| 저작권 | 스톡 시 라이선스 기록; 실사 촬영 시 범위 문서화 *(가정)*. |
