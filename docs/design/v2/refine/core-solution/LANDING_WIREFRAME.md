# Core Solution Landing Page Wireframe

> **Status**: Deprecated  
> **Scope**: Public landing 와이어 — **NOT** Phase 3 메인 앱 `/` 허브. Trinity/public funnel legacy.

> **Superseded by**: [core-solution/DESIGN_PHASE3_MASTER_PLAN.md](./DESIGN_PHASE3_MASTER_PLAN.md) Phase 1 (`/` 시안)

## 1. Global Visual & Layout Rules
- **Color Palette**: Calm Forest (`#3D5246` Primary, `#FAF9F7` Background). 블루/퍼플 그라데이션 절대 금지.
- **Typography**: 넉넉한 여백(Whitespace), 높은 타이포그래피 밀도(채널톡/원티드 수준의 가독성).
- **Tone & Manner**: IT/SaaS/Trinity 용어 배제. 센터 원장님과 실무진이 직관적으로 이해할 수 있는 따뜻하고 신뢰감 있는 한국어 사용.
- **Logo**: Shield Variant 로고 사용.

---

## 2. Section Wireframe

### [Section 1] Header (GNB)
- **Left**: Core Solution 로고 (Shield Variant)
- **Right**: [로그인] [무료로 시작하기] 버튼 (Primary Color `#3D5246`)
- **Style**: 투명한 배경, 스크롤 시 화이트 배경 고정.

### [Section 2] Hero (Product UI Feel)
- **Headline**: "예약부터 회기, 정산까지. 센터 운영의 모든 것을 한곳에서."
- **Sub-copy**: 복잡한 행정 업무는 코어 솔루션에 맡기고, 내담자와의 소중한 시간에 집중하세요.
- **CTA**: [지금 무료로 시작하기] (대형 버튼)
- **Visual**: 실제 대시보드 및 스케줄러 UI 목업 (SVG/클린 UI 형태, 체커보드 PNG 금지). 여백을 넓게 주어 시원한 개방감 제공.

### [Section 3] Features (핵심 기능 3가지)
*배경: 부드러운 오프화이트 (`#FAF9F7`)*
- **Feature 1: 한눈에 들어오는 통합 스케줄러**
  - Copy: "전화, 문자, 방문 예약... 흩어진 일정을 하나의 캘린더로 모았습니다."
  - Visual: 상담사-내담자 매핑이 직관적으로 보이는 스케줄러 UI 요소.
- **Feature 2: 끊김 없는 회기 관리**
  - Copy: "상담 기록부터 다음 예약까지, 내담자의 흐름을 놓치지 마세요."
  - Visual: 회기 진행 상태가 표시된 깔끔한 카드형 UI.
- **Feature 3: 투명하고 정확한 자동 정산**
  - Copy: "복잡했던 급여 계산과 수납 관리를 클릭 몇 번으로 끝내세요."
  - Visual: ERP 기반 정산 내역이 정리된 직관적인 표/그래프 SVG.

### [Section 4] Testimonials (원장님 후기)
*가짜 로고 스트립(Acme 등) 완전 삭제. 실제 현장감 있는 텍스트 후기 중심.*
- **Title**: "이미 많은 센터에서 코어 솔루션과 함께하고 있습니다."
- **Card 1**: "예약 실수도 줄고, 월말 정산 스트레스가 사라졌어요. 오롯이 상담에만 집중할 수 있어 만족합니다." – *따뜻한마음 심리상담센터 원장님*
- **Card 2**: "상담사 선생님들도 일정 확인이 편해졌다고 좋아하세요. 센터 운영이 정말 가벼워졌습니다." – *늘푸른 아동발달센터 실장님*
- **Card 3**: "기존에 쓰던 복잡한 프로그램들과 달리, 딱 필요한 기능만 직관적으로 있어서 금방 적응했습니다." – *포근한 웰니스센터 원장님*

### [Section 5] CTA (Bottom)
- **Headline**: "우리 센터의 든든한 파트너, 지금 바로 경험해보세요."
- **Sub-copy**: 초기 도입비 없이, 센터 규모에 맞는 합리적인 요금제로 시작할 수 있습니다.
- **CTA**: [무료로 시작하기]
- **Note**: Trinity 언급 절대 금지.

### [Section 6] Footer
- **Left**: Core Solution 로고 (회색조/단색 처리), 간단한 회사 정보 (상호, 대표자, 사업자등록번호 등).
- **Right**: 이용약관, 개인정보처리방침, 고객센터 연락처.
- **Style**: 다크 그레이 배경 (`#2C2C2C` 등)에 차분한 텍스트.
