### 📋 표준 컴포넌트 현황

프로젝트에 이미 표준 컴포넌트가 구현되어 있습니다:

#### UI 컴포넌트 (`components/ui/`)
- `Button/` - 표준 버튼 컴포넌트
- `Card/` - 카드 컴포넌트 (ClientCard, ConsultantCard, StatCard 등)
- `Modal/` - 모달 컴포넌트
- `Table/` - 테이블 컴포넌트
- `Form/` - 폼 컴포넌트 (Input, Select, Textarea, Checkbox)
- `Loading/` - 로딩 컴포넌트 (Spinner, ProgressBar, Skeleton)
- `Layout/` - 레이아웃 컴포넌트
- `Icon/` - 아이콘 컴포넌트

#### 공통 컴포넌트 (`components/common/`)
- `CommonPageTemplate` - 공통 페이지 템플릿
- `UnifiedHeader` - 통합 헤더
- `UnifiedLoading` - 통합 로딩
- `MGButton`, `MGCard`, `MGForm`, `MGTable` 등 - MG 시리즈 컴포넌트
