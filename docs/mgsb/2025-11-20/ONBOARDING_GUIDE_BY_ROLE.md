# 역할별 온보딩 가이드

**작성일**: 2025-11-20  
**버전**: 1.0.0  
**상태**: 완료

---

## 📋 개요

역할별 온보딩 후 필수 작업과 권장 작업을 정리한 문서입니다. 각 역할은 다른 온보딩 프로세스를 가지지만, 공통적인 패턴을 공유합니다.

---

## 🎯 레벨별 온보딩 가이드

### 레벨 1: 기본 사용자 온보딩

**온보딩 후 필수 작업:**
- [ ] 프로필 정보 확인/수정
- [ ] 비밀번호 변경 (최초 로그인 시)
- [ ] 알림 설정 확인
- [ ] 서비스 이용 방법 확인

**권장 작업:**
- [ ] 대시보드 기능 탐색
- [ ] 서비스 예약/신청 테스트
- [ ] 내역 조회 기능 확인

---

### 레벨 2: 서비스 제공자 온보딩

**온보딩 후 필수 작업:**
- [ ] 프로필 정보 확인/수정
- [ ] 비밀번호 변경 (최초 로그인 시)
- [ ] 일정 관리 설정
  - [ ] 상담 가능 시간 설정
  - [ ] 반복 일정 설정 (필요시)
- [ ] 할당된 사용자 확인

**권장 작업:**
- [ ] 일정 관리 기능 탐색
- [ ] 기록 작성 기능 확인
- [ ] 통계 기능 확인

---

### 레벨 3: 관리자 온보딩

**온보딩 후 필수 작업:**
- [ ] 비밀번호 변경 (최초 로그인 시)
- [ ] 기본 역할 확인
  - [ ] 역할명 수정 (필요시)
  - [ ] 권한 확인/수정 (필요시)
- [ ] 기본 대시보드 확인
  - [ ] 대시보드 이름 수정 (필요시)
  - [ ] 대시보드 설정 확인
- [ ] 사용자 추가
  - [ ] 서비스 제공자 추가
  - [ ] 기본 사용자 추가
- [ ] 시스템 설정 확인
  - [ ] 공통코드 확인
  - [ ] 알림 설정 확인

**권장 작업:**
- [ ] 사용자 관리 기능 탐색
- [ ] 통계 기능 확인
- [ ] 리포트 기능 확인
- [ ] 시스템 설정 탐색

---

### 레벨 4: 최고 관리자 온보딩

**온보딩 후 필수 작업:**
- [ ] 레벨 3의 모든 필수 작업

  - [ ] 역할 목록 확인
  - [ ] 역할 추가/수정 (필요시)
  - [ ] 권한 설정 확인
- [ ] 시스템 설정
  - [ ] 테넌트 설정 확인
  - [ ] 고급 설정 확인

**권장 작업:**
- [ ] 역할 관리 기능 탐색
- [ ] 시스템 설정 탐색
- [ ] 전체 시스템 기능 확인

## 🔄 온보딩 체크리스트 템플릿

### 테넌트 관리자 온보딩 체크리스트

```
□ 1단계: 최초 로그인
  □ 임시 비밀번호로 로그인
  □ 비밀번호 변경
  □ 프로필 정보 확인/수정

□ 2단계: 기본 설정 확인
  □ 기본 역할 확인
  □ 기본 대시보드 확인
  □ 시스템 설정 확인

□ 3단계: 사용자 추가
  □ 서비스 제공자 추가
  □ 기본 사용자 추가

□ 4단계: 기능 테스트
  □ 주요 기능 테스트
  □ 통계 기능 확인
  □ 리포트 기능 확인

□ 5단계: 커스터마이징
  □ 역할명 수정 (필요시)
  □ 권한 수정 (필요시)
  □ 대시보드 이름 수정 (필요시)
  □ 대시보드 설정 수정 (필요시)
```

---

## ⚠️ 중요 사항

1. **범용 패턴 제공**
   - 이 문서는 범용적인 온보딩 패턴을 제공
   - 실제 입점사는 다른 온보딩 프로세스를 가질 수 있음

2. **MindGarden은 예시**
   - MindGarden의 온보딩 프로세스는 CONSULTATION 업종의 예시
   - 다른 상담소는 다른 온보딩 프로세스를 가질 수 있음

3. **동적 온보딩**
   - 온보딩 프로세스는 업종별로 다름
   - 테넌트 관리자가 온보딩 후 커스터마이징 가능
