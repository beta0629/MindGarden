# 어린이 비디오 찾는 방법

## 무료 스톡 비디오 사이트

### 1. Pexels (추천)
- URL: https://www.pexels.com/search/videos/children%20happy/
- 검색 키워드: "children happy", "kids laughing", "children playing"
- 무료 사용 가능, 상업적 사용 가능
- 다운로드 후 `hero-video.mp4`로 저장

### 2. Pixabay
- URL: https://pixabay.com/videos/search/children%20happy/
- 검색 키워드: "children happy", "kids playing"
- 무료 사용 가능

### 3. Videvo
- URL: https://www.videvo.net/
- 무료 및 유료 옵션 제공

## 비디오 다운로드 및 사용 방법

1. 위 사이트에서 원하는 비디오 검색
2. 비디오 다운로드 (MP4 형식)
3. 파일명을 `hero-video.mp4`로 변경
4. `public/assets/videos/` 폴더에 저장
5. 자동으로 히어로 섹션에 표시됩니다!

## 권장 비디오 내용
- 어린이들이 웃고 즐거워하는 모습
- 편안하고 따뜻한 분위기
- 밝고 긍정적인 느낌
- 해상도: 1920x1080 이상 권장
