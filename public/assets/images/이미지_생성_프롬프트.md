## AI 이미지 생성 도구별 사용법

### 1. Midjourney
- 프롬프트 앞에 `/imagine` 입력
- 설정: `--ar 16:9 --v 6` (16:9 비율, 버전 6)
- 예: `/imagine prompt: [프롬프트] --ar 16:9 --v 6 --style raw`

### 2. DALL-E 3 (ChatGPT)
- ChatGPT에서 프롬프트 입력
- "이미지 생성" 요청
- 설정: 1024x1024 또는 1792x1024

### 3. Stable Diffusion
- 프롬프트 입력
- Negative prompt: "dark, scary, cluttered, messy, text, logo"
- 설정: 1920x1080, steps: 30-50

### 4. Leonardo.ai
- 프롬프트 입력
- Model: Leonardo Diffusion XL
- Aspect Ratio: 16:9
- Resolution: 1024x576 또는 1920x1080

### 5. Ideogram
- 프롬프트 입력
- Style: Realistic 또는 Cinematic
- Aspect Ratio: 16:9

## 이미지 생성 팁

### 공통 설정
- **해상도**: 1920x1080 (또는 16:9 비율)
- **스타일**: Realistic, Cinematic, Warm
- **색상**: Soft pastels (sky blue, peach, lavender, mint)
- **조명**: Natural, soft, warm
- **분위기**: Peaceful, safe, welcoming, cozy

### Negative Prompt (제외할 요소)
```
dark, scary, cluttered, messy, text, logo, watermark, people's faces clearly visible, crowded, chaotic, cold lighting, harsh shadows, medical equipment, clinical atmosphere
```

### 색상 팔레트 참고
- 하늘색: #B8D4E3
- 피치: #FFD4B8
- 라벤더: #E8D5E3
- 민트: #D4F0E8
- 옐로우: #FFF4D4

## 이미지 사용 방법

### 갤러리 이미지
1. 생성된 이미지 다운로드
2. 파일명: `gallery_1.png`, `gallery_2.png`, `gallery_3.png`, `gallery_4.png`
3. 위치: `public/assets/images/` 폴더에 저장
4. 자동으로 갤러리 마퀴에 표시됩니다

### 히어로 포스터
1. 생성된 이미지 다운로드
2. 파일명: `hero.png`
3. 위치: `public/assets/images/hero.png`
4. 비디오 로딩 중 표시됩니다

## 이미지 개수 추천
