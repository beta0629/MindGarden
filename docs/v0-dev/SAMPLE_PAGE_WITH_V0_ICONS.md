```
Create a modern user group icon for a counseling management system:

- 2px stroke width
```

### 2. 아이콘 일관성 유지
- 모든 아이콘에 동일한 stroke width 적용
- 색상 팔레트를 올리브 그린으로 통일
- 크기 체계를 일관되게 적용
- 애니메이션 패턴을 통일

### 3. 접근성 고려사항
- 충분한 대비율 확보
- 의미있는 alt 텍스트 제공
- 키보드 네비게이션 지원
- 스크린 리더 호환성
