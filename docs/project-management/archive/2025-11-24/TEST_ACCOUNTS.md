## 🔐 비밀번호 정보

### 비밀번호 저장 방식
- 모든 비밀번호는 **BCrypt 해시**로 데이터베이스에 저장됩니다.
- 실제 비밀번호는 온보딩 요청의 `checklistJson`에 평문으로 저장되어 있습니다.

### 비밀번호 확인 방법
```sql
-- 온보딩 요청에서 비밀번호 확인
SELECT 
    id,
    tenant_id,
    requested_by,
    checklist_json
FROM onboarding_requests
WHERE requested_by = 'test-category1763957121@example.com'
ORDER BY created_at DESC
LIMIT 1;
```

`checklist_json` 필드에서 `adminPassword` 값을 확인할 수 있습니다.
