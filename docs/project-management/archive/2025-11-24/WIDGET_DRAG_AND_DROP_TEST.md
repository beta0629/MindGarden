## 🖱️ 드래그 앤 드롭 테스트

### 1. 위젯 추가

**방법 1: 위젯 목록에서 추가**
1. "위젯 추가" 버튼 클릭
2. 위젯 타입 선택 (예: "환영 위젯", "통계 위젯")
3. 위젯이 레이아웃에 추가됨

**방법 2: 드래그 앤 드롭으로 추가**
1. 위젯 목록에서 위젯을 드래그
2. 레이아웃 영역으로 드롭
3. 위젯이 배치됨

---

### 2. 위젯 순서 변경

1. 레이아웃에 배치된 위젯의 **드래그 핸들** (⋮⋮ 아이콘) 클릭
2. 원하는 위치로 드래그
3. 다른 위젯과 위치 교환 가능
4. 순서 변경 시 애니메이션 확인

**확인 사항**:
- [ ] 드래그 핸들이 표시됨
- [ ] 드래그 중 위젯이 반투명하게 표시됨 (ghost 효과)
- [ ] 드롭 위치에 미리보기 표시
- [ ] 순서 변경이 즉시 반영됨

---

### 3. 위젯 위치 변경 (그리드 레이아웃)

1. 위젯을 드래그하여 다른 그리드 셀로 이동
2. 위젯 크기 조정 (span 설정)
3. 레이아웃 그리드에 맞춰 정렬

**확인 사항**:
- [ ] 위젯이 그리드 셀에 맞춰 정렬됨
- [ ] 위젯 크기가 자동으로 조정됨
- [ ] 다른 위젯과 겹치지 않음

---

### 4. 위젯 삭제

1. 위젯의 "삭제" 버튼 (🗑️ 아이콘) 클릭
2. 확인 대화상자에서 확인
3. 위젯이 레이아웃에서 제거됨

**확인 사항**:
- [ ] 삭제 버튼이 표시됨
- [ ] 삭제 확인 대화상자 표시
- [ ] 삭제 후 레이아웃이 자동으로 재정렬됨

---

### 5. 위젯 설정 변경

1. 위젯의 "설정" 버튼 (⚙️ 아이콘) 클릭
2. 위젯 설정 모달 열림
3. 설정 값 변경
4. "저장" 버튼 클릭

**확인 사항**:
- [ ] 설정 모달이 정상적으로 열림
- [ ] 설정 값이 저장됨
- [ ] 변경된 설정이 위젯에 반영됨

---

### 6. 대시보드 저장

1. "저장" 버튼 클릭
2. 저장 성공 메시지 확인
3. 대시보드 목록으로 돌아감

**확인 사항**:
- [ ] 저장 성공 메시지 표시
- [ ] 대시보드 목록에 새 대시보드 추가됨 (생성 시)
- [ ] 대시보드 정보가 업데이트됨 (수정 시)

---

### 7. 대시보드 확인

1. 대시보드 목록에서 생성/수정한 대시보드 선택
2. "보기" 버튼 클릭 또는 대시보드 접속
3. 위젯이 정상적으로 표시되는지 확인

**확인 사항**:
- [ ] 위젯이 저장된 순서대로 표시됨
- [ ] 위젯이 저장된 위치에 표시됨
- [ ] 위젯이 정상적으로 동작함

## 🐛 문제 발생 시

### 문제 1: 드래그가 안 됨
- 브라우저 콘솔에서 오류 확인
- `react-sortablejs` 라이브러리 로드 확인
- 드래그 핸들이 표시되는지 확인

### 문제 2: 위젯이 저장되지 않음
- 네트워크 탭에서 API 요청 확인
- 서버 응답 확인
- 브라우저 콘솔 오류 확인

### 문제 3: 위젯이 표시되지 않음
- 대시보드 설정 JSON 확인
- 위젯 타입이 올바른지 확인
- `WidgetRegistry`에 위젯이 등록되어 있는지 확인
