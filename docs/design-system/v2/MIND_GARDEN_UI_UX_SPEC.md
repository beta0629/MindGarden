# 마음 정원 (MindGarden) UI/UX 스펙

> **작성일**: 2026-05-13
> **작성자**: core-designer
> **설명**: 상담 및 웰니스 과제 수행을 통해 개인 정원이 성장하는 시각적 보상(게임화) 시스템 스펙. `core-coder`의 Phase 4 구현 기준.

## 1. 디자인 원칙 및 테마
- **비경쟁/비순위**: 타인의 정원과 비교하는 UI/UX 절대 없음. 리더보드 없음.
- **안전한 보상**: 미접속에 대한 페널티(시듦, 망가짐) 없음. 휴식이 권장되는 플랫폼.
- **주조색 적용**: 배경은 `var(--mg-client-bg-main)`(크림색), 식물 요소는 따뜻하고 부드러운 파스텔 톤 일러스트 (플랫 디자인 기반, 외곽선 없음).

## 2. 화면 IA (Information Architecture)

### 2.1 내담자 (Client)
```text
홈 (피드) / 웰니스 (탭)
└── 마음 정원 위젯 (작은 미리보기)
    └── 정원 상세 씬 (전체 화면 또는 큰 카드)
        ├── 정원 일러스트 영역 (현재 레벨/단계별 시각화)
        ├── 환경/날씨 효과 (잔잔한 빛, 떠다니는 입자)
        ├── 성장 바 (Progress Bar) - 다음 단계까지 남은 이벤트
        └── 최근 기록 / 영양분 획득 기록
```

## 3. 시각화 (Garden Scene) 및 성장 단계

### 3.1 성장 단계 정의 (Growth Stages)
- **1단계 (씨앗)**: 부드러운 흙 위에 작은 씨앗, 따뜻한 톤.
- **2단계 (새싹)**: 작은 연두색 잎 두 개 (`var(--mg-client-primary-light)` 계열).
- **3단계 (자라나는 식물)**: 잎이 많아지고 줄기가 튼튼해짐.
- **4단계 (꽃/열매)**: 코랄(`var(--mg-client-primary)`) 색상의 예쁜 꽃이 피거나 열매가 맺힘.
*(각 단계 전환 시 파티클/페이드 애니메이션 적용)*

### 3.2 컴포넌트 스펙 (Organisms)
- **`GardenScene`**
  - **영역**: 상단 60% 비중.
  - **배경**: 은은한 그라데이션 (예: 하늘색 → 크림색).
  - **에셋**: 단계별 SVG 일러스트 또는 정적 이미지.
  - **효과**: CSS/Reanimated 기반의 부드러운 떠다니는 빛무리(오퍼시티 10~20%).
- **`GrowthProgressBar`**
  - 하단 배치. "다음 변화까지 1걸음" (횟수 명시 지양 시 "물주기 완료" 등으로 대체 가능).
  - 바 색상: `var(--mg-client-primary)`, 트랙 색상: `var(--mg-gray-100)`.

## 4. 와이어프레임 및 상태별 화면

### 4.1 Welcome / Onboarding State
- **비주얼**: 빈 화분이나 부드러운 흙 바닥.
- **카피**: "나만의 마음 정원이에요. 상담을 마치거나 마음챙김을 할 때마다 조금씩 자라납니다."
- **액션**: "첫 물주기 (일기 쓰기)" CTA 버튼.

### 4.2 성장 이벤트 (Level Up / Growth Event)
- **트리거**: 상담 완료 시, 심리 검사 완료 시 등.
- **애니메이션**: 물뿌리개 일러스트 또는 빛이 내리는 효과 → 식물 일러스트가 다음 단계로 페이드 인(Fade-in) 또는 스케일업(Scale-up).
- **토스트**: "마음 정원이 한 뼘 자랐어요 🌱" (`var(--mg-color-success)` 아이콘 포함).

### 4.3 주간 상한 도달 (Weekly Cap Reached)
- **트리거**: 과도한 앱 사용(집착)을 방지하기 위한 성장 제한 시점.
- **비주얼**: 정원 위에 부드러운 햇살 또는 달빛(저녁) 효과.
- **카피**: "이번 주 정원은 충분한 햇살을 받았어요. 편안한 마음으로 쉬어가세요."

## 5. 접근성 (Accessibility) 및 설정

### 5.1 애니메이션 감소 (Reduced Motion)
- 운영체제 설정(Reduce Motion) 감지 또는 앱 내 설정 반영.
- **ON 시**: 성장 애니메이션, 파티클(떠다니는 빛), 스와이프 트랜지션 무효화.
- 상태 변화는 애니메이션 없이 즉각적인 이미지 교체(Cross-fade 100ms)로 처리.

### 5.2 스크린 리더 (Screen Reader)
- `GardenScene`의 `accessibilityLabel`: "당신의 마음 정원입니다. 현재 2단계인 새싹 상태입니다."
- `GrowthProgressBar`의 `accessibilityValue`: 퍼센트 혹은 남은 단계 수 지원.

## 6. 상담사 앱 노출 정책 (옵트인 무관)
- **원칙**: 내담자의 "정원 레벨/성장치"는 기본적으로 상담사에게 노출하지 않음 (치료적 개입에 불필요한 편견 방지).
- **예외**: 상담 일지 내담자 활동 요약에 "이번 주 웰니스 과제 참여도: 높음" 형태의 텍스트 배지로만 변환하여 제공. 정원 그래픽 자체는 철저히 **내담자의 개인 공간**으로 격리.
