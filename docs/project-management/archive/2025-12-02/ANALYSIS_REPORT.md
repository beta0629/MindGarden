## 통합 워크플로우 요구사항

### 핵심 프로세스
1. **상담사 등록** → 상담사 프로필 생성, 전문분야 설정
2. **내담자 등록** → 내담자 정보 입력, 상담 요구사항 파악  
3. **매칭** → 상담사-내담자 자동/수동 매칭
4. **회기관리** → 상담 회기 계획 수립
5. **스케줄** → 구체적인 상담 일정 관리

### 데이터 연동 포인트
- 상담사 전문분야 ↔ 내담자 상담 요구사항
- 매칭 정보 ↔ 회기 계획
- 회기 정보 ↔ 스케줄 데이터
- 모든 단계에서 tenantId 기반 멀티테넌시 적용
