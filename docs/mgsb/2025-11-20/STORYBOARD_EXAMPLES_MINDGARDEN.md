# MindGarden 스토리보드 예시

**작성일**: 2025-11-20  
**버전**: 1.0.0  
**상태**: 완료

---

## 📋 개요

MindGarden은 CONSULTATION 업종의 실제 구현 예시입니다. 이 문서는 MindGarden의 주요 사용자 시나리오와 화면 흐름을 스토리보드로 정리한 것입니다.

**⚠️ 참고**: MindGarden의 역할은 CONSULTATION 업종의 예시일 뿐이며, 다른 상담소는 다른 역할 구조를 가질 수 있습니다.

---

## 👤 역할별 스토리보드

### 1. 내담자 (CLIENT) 스토리보드

#### 시나리오 1: 상담 예약

**화면 흐름:**
```
1. 로그인 화면
   - 이메일/비밀번호 입력
   - 소셜 로그인 (카카오, 네이버, 구글)
   ↓
2. 내담자 대시보드
   - 내 상담 내역 요약
   - 빠른 예약 버튼
   - 최근 상담 기록
   ↓
3. 상담사 선택
   - 상담사 목록
   - 상담사 프로필
   - 전문분야 필터
   ↓
4. 일정 선택
   - 캘린더 표시
   - 가능한 시간대 표시
   - 시간 선택
   ↓
5. 예약 정보 입력
   - 상담 목적
   - 특이사항
   - 예약 확인
   ↓
6. 결제 (필요시)
   - 결제 수단 선택
   - 결제 정보 입력
   - 결제 완료
   ↓
7. 예약 완료
   - 예약 확인 메시지
   - 일정 알림 설정
   - 대시보드로 이동
```

**인터랙션:**
- 캘린더에서 날짜/시간 선택
- 드롭다운에서 상담사 선택
- 텍스트 입력으로 상담 목적 입력
- 결제 버튼 클릭

---

#### 시나리오 2: 상담 내역 조회

**화면 흐름:**
```
1. 내담자 대시보드
   ↓
2. 상담 내역 메뉴
   ↓
3. 상담 내역 목록
   - 날짜별 정렬
   - 상태별 필터 (예정, 완료, 취소)
   ↓
4. 상담 내역 상세
   - 상담사 정보
   - 상담 일시
   - 상담 메시지 내역
   - 상담 기록 (상담사가 작성한 경우)
   ↓
5. 상담 메시지 보기
   - 메시지 목록
   - 메시지 입력 (진행 중인 상담)
```

**인터랙션:**
- 테이블에서 상담 내역 클릭
- 필터로 상태별 조회
- 메시지 입력창에 텍스트 입력

---

### 2. 상담사 (CONSULTANT) 스토리보드

#### 시나리오 1: 일정 관리

**화면 흐름:**
```
1. 로그인 화면
   ↓
2. 상담사 대시보드
   - 오늘의 일정
   - 할당된 내담자 수
   - 이번 달 상담 통계
   ↓
3. 일정 관리 메뉴
   ↓
4. 일정 목록/캘린더
   - 월별/주별/일별 보기
   - 일정 추가 버튼
   ↓
5. 일정 추가
   - 날짜/시간 선택
   - 일정 유형 선택 (상담, 회의, 휴무 등)
   - 반복 일정 설정 (선택)
   ↓
6. 일정 저장
   ↓
7. 일정 목록 업데이트
```

**인터랙션:**
- 캘린더에서 날짜 클릭하여 일정 추가
- 드래그 앤 드롭으로 일정 이동
- 일정 클릭하여 수정/삭제

---

#### 시나리오 2: 상담 기록 작성

**화면 흐름:**
```
1. 상담사 대시보드
   ↓
2. 오늘의 일정
   - 예정된 상담 목록
   ↓
3. 상담 시작
   - 상담 시작 버튼
   ↓
4. 상담 진행
   - 내담자 정보
   - 상담 메시지 입력
   - 상담 기록 작성
   ↓
5. 상담 완료
   - 상담 요약 입력
   - 상담 기록 저장
   ↓
6. 상담 기록 확인
   - 저장된 상담 기록
   - 내담자에게 공개 여부 설정
```

**인터랙션:**
- 상담 시작 버튼 클릭
- 텍스트 에디터로 상담 기록 작성
- 체크박스로 공개 여부 설정

---

### 3. 관리자 (ADMIN) 스토리보드

#### 시나리오 1: 사용자 관리

**화면 흐름:**
```
1. 로그인 화면
   ↓
2. 관리자 대시보드
   - 전체 통계
   - 최근 활동
   - 빠른 액션
   ↓
3. 사용자 관리 메뉴
   ↓
4. 사용자 목록
   - 사용자 검색
   - 역할별 필터
   - 지점별 필터
   ↓
5. 사용자 추가
   - 사용자 정보 입력
   - 역할 선택
   - 지점 할당
   ↓
6. 사용자 저장
   ↓
7. 사용자 목록 업데이트
```

**인터랙션:**
- 검색창에 이름/이메일 입력
- 필터 드롭다운에서 역할/지점 선택
- 사용자 추가 버튼 클릭
- 모달에서 사용자 정보 입력

---

#### 시나리오 2: 공통 코드 관리

**화면 흐름:**
```
1. 관리자 대시보드
   ↓
2. 설정 메뉴
   ↓
3. 공통 코드 관리 메뉴
   ↓
4. 코드 그룹 선택
   ├─ CoreSolution 공통 코드 (읽기 전용)
   └─ MindGarden 공통 코드 (읽기/쓰기)
   ↓
5. 코드 목록 조회
   - 코드 그룹별 필터
   - 코드 타입별 필터 (CORE/TENANT)
   ↓
6. 코드 추가/수정/삭제
   - 코드 추가 (MindGarden 코드만)
   - 코드 수정 (MindGarden 코드만)
   - 코드 삭제 (MindGarden 코드만)
   ↓
7. 코드 정보 입력
   - 코드 값
   - 코드 라벨
   - 한글명 (필수)
   - 설명
   - 정렬 순서
   ↓
8. 코드 저장
```

**인터랙션:**
- 코드 그룹 드롭다운 선택
- 코드 타입 필터 (CORE/TENANT)
- 코드 추가 버튼 클릭
- 코드 정보 입력 폼
- 저장 버튼 클릭

**권한:**
- CoreSolution 공통 코드: 읽기 전용 (HQ Admin만 수정 가능)
- MindGarden 공통 코드: 읽기/쓰기 (Tenant Admin 수정 가능)

---

#### 시나리오 2: 통계 조회

**화면 흐름:**
```
1. 관리자 대시보드
   ↓
2. 통계 메뉴
   ↓
3. 통계 대시보드
   - 기간 선택
   - 통계 카테고리 선택
   ↓
4. 통계 데이터 표시
   - 차트/그래프
   - 테이블 데이터
   ↓
5. 리포트 다운로드 (선택)
   - 엑셀 다운로드
   - PDF 다운로드
```

**인터랙션:**
- 날짜 범위 선택기로 기간 선택
- 탭으로 통계 카테고리 전환
- 차트 클릭하여 상세 정보 표시
- 다운로드 버튼 클릭

**화면 흐름:**
```
1. 로그인 화면
   ↓

   - 전체 테넌트 통계

   ↓
3. 테넌트 관리 메뉴
   ↓

   - 테넌트 검색
   - 업종별 필터
   - 상태별 필터
```

## 🔄 주요 화면 전환

### 내담자 화면 전환

```
대시보드
  ├─ 상담 예약 → 상담사 선택 → 일정 선택 → 예약 완료
  ├─ 상담 내역 → 상담 상세 → 메시지 보기
  ├─ 결제 내역 → 결제 상세
  └─ 프로필 설정
```

### 상담사 화면 전환

```
대시보드
  ├─ 일정 관리 → 일정 추가/수정
  ├─ 할당된 내담자 → 내담자 상세 → 상담 기록 작성
  ├─ 상담 통계
  └─ 프로필 설정
```

### 관리자 화면 전환

```
대시보드
  ├─ 사용자 관리 → 사용자 추가/수정 → 역할 할당
  ├─ 통계 → 통계 대시보드 → 리포트 다운로드
  ├─ 일정 관리 → 전체 일정 조회
  ├─ 결제 관리 → 결제 내역 조회
  └─ 설정 관리 → 시스템 설정
```

## 📱 반응형 화면 구성

### 데스크탑 (1920px 이상)

- 사이드바 네비게이션
- 메인 콘텐츠 영역 (넓은 레이아웃)
- 다중 컬럼 통계 카드

### 태블릿 (768px ~ 1919px)

- 햄버거 메뉴
- 메인 콘텐츠 영역 (2컬럼 레이아웃)
- 카드 형태의 통계

### 모바일 (767px 이하)

- 하단 네비게이션
- 메인 콘텐츠 영역 (1컬럼 레이아웃)
- 리스트 형태의 통계

## 🎨 주요 UI 컴포넌트

### 대시보드 위젯

- **통계 카드**: 숫자, 아이콘, 변화율
- **일정 위젯**: 오늘의 일정, 캘린더
- **최근 활동**: 최근 항목 목록
- **빠른 액션**: 자주 사용하는 기능 버튼

### 폼 컴포넌트

- **입력 필드**: 텍스트, 이메일, 전화번호
- **선택 필드**: 드롭다운, 라디오, 체크박스
- **날짜/시간 선택**: 캘린더, 시간 선택기
- **파일 업로드**: 이미지, 문서

### 테이블 컴포넌트

- **정렬**: 컬럼 헤더 클릭
- **필터링**: 검색창, 필터 드롭다운
- **페이징**: 페이지 번호, 페이지 크기 선택
- **액션**: 수정, 삭제, 상세 보기 버튼

---

## ⚠️ 중요 사항

1. **MindGarden은 예시일 뿐**
   - MindGarden의 화면 흐름은 CONSULTATION 업종의 예시
   - 다른 상담소는 다른 화면 구조를 가질 수 있음

2. **동적 역할 시스템**
   - 역할은 동적으로 생성되므로 화면도 동적으로 구성
   - 역할별 대시보드는 테넌트 관리자가 커스터마이징 가능

3. **업종별 차이**
   - 학원: 강좌 신청, 출석 체크, 성적 조회
   - 상담소: 상담 예약, 상담 기록, 결제
   - 기타: 업종별 다른 화면 흐름
