# 제품 로드맵 (CoreSolution, 2026~2028)

## 1) 로드맵 원칙(표준 기반)

- **표준화 우선**: 공통코드/동적 조회/하드코딩 금지 원칙 준수
- **보안·개인정보 선행**: 암호화/권한/감사/운영 표준을 제품 기능에 내재화
- **업종 확장 가능 구조**: 상담→교육→1인 서비스업으로 확장 가능한 공통 흐름 유지
- **AI는 “보조/검증/근거” 중심**: 오판 방어·사람 검수·버전 추적을 기본값으로 설계

## 2) 2026(1차년도): 상용화 준비 + 심리검사 AI 실증

### 플랫폼 공통
- 온보딩/테넌트/권한 표준 고도화 및 운영 안정화
- 관리자 대시보드 데이터 표준화(지표 정의, 저장/조회 규칙)
- CI/CD 안정화(배포 표준 준수), 모니터링/알림 표준 적용 강화

### AI(심리검사 리포트) — TCI/MMPI 우선
- 스캔 PDF 업로드→암호화 저장→추출/검증→리포트 생성 MVP 운영
- 병원/센터별 양식 차이를 고려한 **템플릿 감지 + 추출 고도화** 단계 착수
- 리포트 품질 지표(검증 실패율/사람 검수율/수정율) 수집 및 개선 루프 구축

## 3) 2027(2차년도): 업종 확장 + 데이터 기반 운영 자동화

- 교육(학원) 모듈 확장: 다지점/정산/권한/리포트 표준화
- 운영 자동화: 정산 이상 징후 탐지, 이탈/재참여 추천 등(데이터 기반)
- 고객 성공(CS) 체계: 온보딩/교육 콘텐츠/헬프데스크/SLA

## 4) 2028(3차년도): 대규모 확장 + 고도화

- 1인 서비스업(IT센터 입주사 등) 확장 템플릿 제공
- 보안/운영 인증 준비(내부 기준에 맞춘 정책/절차 고도화)
- AI 파트 고도화: “설명가능성(evidence)” 강화, 위험도/품질 자동 분류 정밀화

## 5) 핵심 KPI(예시)

- **MRR/유료 테넌트 수**, **활성 사용자 수**
- **업무 자동화율**(수기 업무 감소), **정산 오류율**
- 심리검사 AI: **검증 통과율**, **사람 검수 비율**, **수정 후 수용률**
