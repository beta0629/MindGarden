## 1. 수정공고 핵심 요약

### 1.1 선정 규모 축소 및 커트라인 상향
| 항목 | 내용 |
|------|------|
| **선정 인원** | 전년 ~780명 → **2026년 ~300명** (일반 110명, 특화 190명) 수준으로 대폭 축소 |
| **평가 커트라인** | 서류·발표 각 단계 **60점 → 70점**으로 상향. 70점 미만 시 해당 단계에서 즉시 탈락 |
| **전략적 함의** | "좋은 아이디어" 수준을 넘어 **타겟 고객 명확 + 수익 모델(BM) 탄탄 + 압축적 사업계획서(PSST)** 필요 |

### 1.2 사업화 자금 2단계 서바이벌 차등 지원 (가장 중요)
| 구분 | 내용 |
|------|------|
| **1단계** | 약 **2,000만 원** 선지급 — 초기 시장조사 및 **MVP 제작**용 |
| **2단계** | 협약 중간 시점에 **1단계 성과(MVP 진척도 등) 평가** → 전체 팀 중 **상위 약 50%** 내외만 추가 약 2,000만 원 지급 |
| **전략적 포인트** | B2B 멀티테넌트 SaaS는 "전체 완성"을 노리다 기한을 놓치면 2단계 수급 실패. **1단계 평가 시점에 원장/상담사/내담자 핵심 대시보드·기본 예약·과금 등 시연 가능한 MVP(React+Java 연동)**를 집중해 실현 가능성을 증명하는 전략이 필수 |

### 1.3 지원금 사용처 및 인건비 제한
| 항목 | 내용 |
|------|------|
| **제한** | **배우자 및 직계가족 인건비**는 지원금(사업비)으로 **엄격히 제한** |
| **전략적 포인트** | 지원금은 **서버·클라우드 인프라, 디자인 외주, 초기 마케팅** 등으로 전액 배정. 인건비는 **비가족 채용**에 한해 집행 가능 |

## 2. 문서·PSST 반영 방향 (실제 사업 가능성 강조)

### 2.1 70점 커트라인 대응 — 압축적·증거 기반 PSST
- **타겟 고객 명확화**: 1차(상담센터)·2차(학원)·3차(1인 기업) 단계별 수치화 유지, "누구에게 팔 것인가" 한 페이지 내 요약 가능하도록 정리.
- **수익 모델(BM) 탄탄성**: SaaS 구독(MRR) + 부가서비스 구조, 3개년 매출·손익분기점(2027년), PoC→유료 전환율 등 **수치로 증명**.
- **압축적 분량**: 서류평가에서 핵심만 빠르게 읽히도록 요약·표·불릿 활용, 중복 설명 축소.

### 2.2 2단계 자금 수급 대응 — 1단계 MVP 집중
- **1단계 목표 명시**: "1단계 평가 시점까지 시연 가능한 MVP"를 문서에 명확히 기술.
  - 원장(Admin)·상담사(Consultant)·내담자(Client) **핵심 대시보드** (이미 구현됨 → 안정화·데모 시나리오 정리).
  - **기본 예약·과금(결제/정산)** 플로우 동작 시연.
  - React 프론트엔드 + Java 백엔드 연동, GitHub CI/CD 기반 배포로 **실제 동작 증명**.
- **전체 기능을 한 번에 완성**하려는 표현 제거, **"1단계에서는 검증용 MVP, 2단계 수급 후 확장"** 구조로 서술.
- **로드맵 시각화**: 협약 시점(6월) 기준으로 **8개월 분기별 계획 재정렬** — 1단계(6~9월) 핵심 대시보드·결제 모듈 안정화 및 중간평가 시연 준비, 2단계(10~1월) AI 고도화 및 유료 전환 실적 확보. 제출 PDF에는 **동작 중인 관리자 화면 스크린샷** 포함으로 "선정 즉시 사업화 가능" 인상 강화.

### 2.3 지원금 사용 계획 — 사용처·제한 명시
- **배우자·직계가족 인건비**: 지원금으로 **집행 불가**임을 명시. (팀 구성표에서도 "지원금으로 충원하는 인력 = 비가족"만 기재.)
- **지원금 비목**:  
  - **인프라**: 서버·클라우드·모니터링 구축  
  - **외주**: 보안 점검, 디자인 외주(필요 시)  
  - **실증/사업화**: PoC 운영, 초기 마케팅  
  - **인건비**: 비가족 채용(추가 개발자 등)에 한해 집행  
- 사용처 표에 "배우자·직계가족 인건비 제외" 각주 또는 문구 추가.

## 4. 결론

- **선정 300명·70점 커트라인**: 타겟 고객 명확, BM 탄탄, **압축적·증거 기반 PSST**로 서류/발표 모두 70점 이상을 노린다.
- **2단계 차등 지원**: 1단계에 **원장/상담사/내담자 대시보드 + 기본 예약·과금** 시연 가능한 MVP를 집중해, 2단계 상위 50% 진입을 확보한다.
- **지원금 사용**: 배우자·직계가족 인건비는 지원금 집행 불가. 인프라·외주·마케팅·비가족 인건비로 전액 배정 계획을 명시한다.

이 방향으로 문서를 다듬으면 "실제 사업이 가능한 것만 뽑는다"는 2026년 선정 기준에 맞춘 제출이 가능하다.
