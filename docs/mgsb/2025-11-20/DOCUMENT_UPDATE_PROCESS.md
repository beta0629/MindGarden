# 문서 업데이트 프로세스

**작성일**: 2025-11-20  
**목적**: 매일 날짜별 폴더로 문서 이동 및 업데이트 프로세스 정의

---

## 📋 업데이트 대상 문서

다음 문서들은 매일 새로운 날짜 폴더로 이동되면서 업데이트되어야 합니다:

1. **`PENDING_DEVELOPMENT_ITEMS.md`** - 미개발 항목 정리 및 우선순위
2. **`DEVELOPMENT_CHECKLIST.md`** - 개발 체크리스트

---

## 🔄 매일 업데이트 프로세스

### Step 1: 새 날짜 폴더 생성

매일 작업 시작 전, 오늘 날짜 폴더를 생성합니다.

**폴더 구조**:
```
docs/mgsb/
├── 2025-11-20/          # 어제 날짜
│   ├── PENDING_DEVELOPMENT_ITEMS.md
│   ├── DEVELOPMENT_CHECKLIST.md
│   └── ...
└── 2025-11-21/          # 오늘 날짜 (새로 생성)
    ├── PENDING_DEVELOPMENT_ITEMS.md
    ├── DEVELOPMENT_CHECKLIST.md
    └── ...
```

### Step 2: 문서 복사 및 업데이트

#### 2.1 PENDING_DEVELOPMENT_ITEMS.md 업데이트

**업데이트 항목**:
- [ ] 문서 상단 날짜 업데이트
  - `작성일`: 오늘 날짜로 변경
  - `최종 업데이트`: 오늘 날짜로 변경
- [ ] 완료된 작업 체크
  - 완료된 항목은 `- [ ]` → `- [x]`로 변경
  - 완료된 작업을 "완료된 작업 ✅" 섹션으로 이동
- [ ] 진행 중인 작업 상태 업데이트
  - 진행률 업데이트
  - 상태 변경 (🚧 진행 중 → ✅ 완료 등)
- [ ] 새로운 작업 추가 (필요 시)
- [ ] 우선순위 재조정 (필요 시)

**예시**:
```markdown
**작성일**: 2025-11-21  # 어제: 2025-11-20
**최종 업데이트**: 2025-11-21
```

#### 2.2 DEVELOPMENT_CHECKLIST.md 업데이트

**업데이트 항목**:
- [ ] 문서 상단 날짜 업데이트
  - `작성일`: 오늘 날짜로 변경
  - `최종 업데이트`: 오늘 날짜로 변경
- [ ] 완료된 체크리스트 항목 체크
  - `- [ ]` → `- [x]`로 변경
- [ ] 담당자 및 완료일 업데이트
  - 완료된 항목의 `담당자` 및 `완료일` 필드 채우기
- [ ] 진행 상황 요약 업데이트
  - P0, P1, P2 완료 항목 수 업데이트

**예시**:
```markdown
**작성일**: 2025-11-21
**최종 업데이트**: 2025-11-21

### 1. 백엔드: Ops 포털 인증/권한 강화
**예상 시간**: 0.5일  
**담당자**: 홍길동  
**완료일**: 2025-11-21

#### 체크리스트
- [x] `DashboardOpsController` 권한 체크 추가  # 완료
- [ ] `PricingPlanOpsController` 권한 체크 추가  # 진행 중
```

### Step 3: 이전 날짜 폴더 정리 (선택)

이전 날짜 폴더는 보관하되, 최신 상태는 항상 오늘 날짜 폴더에 있습니다.

## 🔧 자동화 스크립트 (선택)

매일 수동으로 하는 것이 번거로우면 스크립트를 사용할 수 있습니다.

### 스크립트 위치
`scripts/update-daily-documents.sh`

### 사용법
```bash
# 오늘 날짜로 새 폴더 생성 및 문서 복사
./scripts/update-daily-documents.sh

# 특정 날짜 지정
./scripts/update-daily-documents.sh 2025-11-21
```

---

## 📌 중요 사항

1. **항상 오늘 날짜 폴더에서 작업**: 최신 상태는 항상 오늘 날짜 폴더에 유지
2. **완료 즉시 업데이트**: 작업 완료 시 바로 체크리스트 업데이트
3. **날짜 일관성**: 모든 문서의 날짜는 동일하게 유지
4. **이전 폴더 보관**: 이전 날짜 폴더는 삭제하지 않고 보관 (히스토리)

## 📊 진행 상황 추적

### 일일 진행률 계산

```
일일 진행률 = (오늘 완료된 항목 수 / 전체 항목 수) × 100
```

### 주간 진행률 계산

```
주간 진행률 = (이번 주 완료된 항목 수 / 이번 주 계획 항목 수) × 100
```
