# CoreSolution 문서 디렉토리

## 📁 폴더 구조

### 날짜별 폴더
- `2025-01/`: 2025년 1월 문서 (월별 정리)
- `YYYY-MM-DD/`: 일별 작업 폴더 (오늘 할 일 체크리스트)
  - 예: `2025-01-XX/TODAY_TODO_CHECKLIST.md`
- `archive/`: 오래된 문서 보관

## 📋 문서 정리 규칙

1. **날짜별 폴더**: 월별로 폴더 생성 (예: `2025-01/`, `2025-02/`)
2. **현재 작업 문서**: 최신 월 폴더에 배치
3. **오래된 문서**: `archive/` 폴더로 이동
4. **마스터 문서**: 항상 최신 폴더에 유지

## 📝 문서 업데이트 가이드

1. **일별 작업**: 오늘 날짜 폴더에 `TODAY_TODO_CHECKLIST.md` 생성/업데이트
2. **월별 문서**: 현재 월 폴더(예: `2025-01/`)에 장기 계획 문서 보관
3. **완료된 작업**: 마스터 TODO와 오늘 체크리스트 모두 업데이트
4. **월말 정리**: 완료된 일별 폴더는 archive로 이동 (선택적)

## 📋 일별 체크리스트 사용법

1. 매일 아침 오늘 날짜 폴더 생성 (자동 또는 수동)
2. `TODAY_TODO_CHECKLIST.md` 파일에서 오늘 할 일 확인
3. 작업 완료 시 체크박스 체크
4. 작업 로그에 완료 시간과 내용 기록
5. 하루 종료 시 진행률 업데이트
