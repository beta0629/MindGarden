## 2. 서비스 목적

마인드가든은 심리상담센터의 운영을 디지털화한 B2B SaaS 플랫폼입니다. 내담자가 심리상담을 예약·진행·관리하는 모바일 앱과, 상담사·관리자가 운영하는 웹/앱 어드민으로 구성됩니다. 네이버 로그인은 내담자가 별도 회원가입 절차 없이 신속하게 서비스를 이용할 수 있도록 지원하기 위한 간편 로그인 수단입니다.

## 3. 수집·이용 항목

### 필수 수집 (네이버 콘솔 [API 설정])
| 항목 | 사용 목적 | 보관 기간 |
|---|---|---|
| 회원이름 (name) | 상담사·관리자에게 노출되는 내담자 표시명 | 회원 탈퇴 시까지 |
| 이메일 주소 (email) | 회원 식별 · 알림 발송 (예약 확정·결제·일정 변경) | 회원 탈퇴 시까지 |
| 휴대전화번호 (mobile) | 상담 일정 SMS·카카오 알림톡 발송, 동일 회원 통합매칭 | 회원 탈퇴 시까지 |

### 미수집 (콘솔에서 체크 해제)
- 생일, 출생연도, 연령대, 성별, 프로필 이미지, 별명, 본인 식별값(CI), 연계정보(DI)
- 사유: 앱 내 표시·사용 흐름이 없어 수집하지 않음

## 3-1. 서비스 적용 형태

| 항목 | 적용 여부 | 사유 |
|---|---|---|
| 네이버 로그인을 통한 신규 회원 가입에 적용 | ☑ 적용 | 내담자가 처음 진입하는 경우 자동으로 「간편 회원가입」 화면으로 이동, 네이버에서 받은 이름·이메일·휴대폰이 자동 입력되며 사용자는 약관 동의 후 가입을 완료합니다. **별도의 비밀번호는 요구하지 않습니다.** |
| 회원가입 없이 단순 로그인 인증에 적용 | ☑ 적용 | 이미 가입된 사용자가 네이버 ID 로 다시 로그인하는 흐름 |
| 게시판 글쓰기/댓글 등 일부 메뉴에 적용 | ☐ 미적용 | 해당 사항 없음 |

## 4. 사용자 흐름

```
[사용자] 모바일 앱 진입
   ↓
[로그인 화면] "네이버로 로그인" 버튼 탭
   ↓
[네이버 인증 화면] OAuth 동의
   ↓
[백엔드] /api/v1/oauth2/social-login (provider=NAVER)
   ↓
   ├─ 기존 회원 → 즉시 로그인
   └─ 신규 회원 → 간편 회원가입 화면 (수집 항목 재확인 + 이용약관 동의) → 가입 완료
```

## 5. 보안·개인정보 처리

- 개인정보 처리방침: https://mindgarden.core-solution.co.kr/privacy (또는 앱 내 약관 화면)
- 이용약관: https://mindgarden.core-solution.co.kr/terms (또는 앱 내 약관 화면)
- 토큰 보관: AES-256 암호화 후 서버 보관, JWT Access Token 은 모바일 SecureStore (iOS Keychain · Android EncryptedSharedPreferences)
- 회원 탈퇴 시 네이버 연결 해제 API 호출 + 개인정보 즉시 파기 (5.1.1(v) 준수)
