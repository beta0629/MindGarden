# 무료 이미지 사용 확인 가이드

## 현재 사용 중인 이미지 확인

현재 `public/assets/images/` 폴더에 있는 이미지들이 무료 이미지인지 확인하는 방법입니다.

## 무료 이미지 사이트 (상업적 사용 가능)

### ✅ 완전 무료 (상업적 사용 가능)
1. **Unsplash** - https://unsplash.com/
   - 모든 이미지 무료 사용 가능
   - 상업적 사용 가능
   - 저작권 표시 불필요

2. **Pexels** - https://www.pexels.com/
   - 모든 이미지 무료 사용 가능
   - 상업적 사용 가능
   - 저작권 표시 불필요

3. **Pixabay** - https://pixabay.com/
   - 모든 이미지 무료 사용 가능
   - 상업적 사용 가능
   - 저작권 표시 불필요

4. **Freepik** (무료 계정)
   - 일부 이미지 무료
   - 상업적 사용 시 라이선스 확인 필요

## 이미지 교체 방법

### 1. 무료 이미지 다운로드
위 사이트에서 ADHD 상담소에 맞는 이미지 검색:
- "counseling room"
- "therapy room"
- "comfortable space"
- "peaceful interior"

### 2. 파일 교체
다운로드한 이미지를 다음 파일명으로 저장:
- `gallery_1.png` → `public/assets/images/gallery_1.png`
- `gallery_2.png` → `public/assets/images/gallery_2.png`
- `gallery_3.png` → `public/assets/images/gallery_3.png`
- `gallery_4.png` → `public/assets/images/gallery_4.png`
- `hero.png` → `public/assets/images/hero.png`

### 3. 자동 반영
파일을 교체하면 자동으로 홈페이지에 반영됩니다!

## 권장 이미지 내용 (ADHD 전문 상담소)
- 편안하고 안정적인 상담실 내부
- 따뜻한 조명의 공간
- 전문적이면서도 편안한 분위기
- ADHD 상담에 적합한 환경

## 라이선스 확인
현재 사용 중인 이미지의 출처를 확인하고, 무료 이미지가 아니라면 위 사이트에서 무료 이미지로 교체하는 것을 권장합니다.
